# Semantic-Guided Multi-Attention Localization for Zero-Shot Learning

Yizhe Zhu, Jianwen Xie, Zhiqiang Tang, Xi Peng, Ahmed Elgammal (2019)

## 🧩 Problem to Solve

기존 객체 분류는 대규모 레이블링된 훈련 데이터에 의존하지만, 실제 세계에는 시각적 데이터가 부족하거나 없는 수많은 객체가 존재한다. Zero-shot learning (ZSL)은 이러한 문제를 해결하기 위해 클래스의 의미론적 표현을 도입하여 보지 못한 클래스를 인식하는 것을 목표로 한다.

현재 Zero-shot learning 접근 방식들은 주로 시각적-의미론적 임베딩을 위한 적절한 매핑 함수를 학습하는 데 중점을 두며, **판별적인 시각적 특징 학습의 중요성을 간과**한다. 특히 CNN 모델은 전통적인 객체 인식 작업을 위해 사전 훈련되었기 때문에, ZSL 작업에 충분히 대표적인 특징을 추출하지 못할 수 있다. 또한, 미세 분류 (fine-grained) 시나리오에서는 전역적 시각 특징(global visual feature)만으로는 클래스 간의 미묘한 차이를 포착하기 어렵다.

이 논문은 **객체의 가장 판별적인 부분(discriminative parts)을 자동으로 발견하여 미세 분류 Zero-shot learning의 성능을 향상시키는 문제**를 해결하고자 한다. 즉, 사람의 어떠한 주석(annotation) 없이도 객체의 핵심적인 부분을 찾아내고 이를 Zero-shot learning에 효과적으로 활용하는 방법을 제안한다.

## ✨ Key Contributions

이 논문의 중심적인 직관 또는 설계 아이디어는 다음과 같다:
Zero-shot learning에서 객체의 전역적 특징(global features)뿐만 아니라, **의미론적 정보에 의해 유도된(semantic-guided) 여러 개의 판별적인 부분(discriminative parts) 특징을 함께 학습**함으로써 시각적 표현의 풍부함과 판별력을 크게 향상시킬 수 있다. 이를 위해, 다중 어텐션(multi-attention) 메커니즘을 도입하여 객체의 핵심 부분을 자동으로 탐지하고, 이 부분 특징과 전체 객체 특징을 결합하여 학습한다. 또한, 임베딩 소프트맥스 손실(embedding softmax loss)과 클래스-중심 삼중항 손실(class-center triplet loss)을 함께 사용하여 학습된 특징이 클래스 간에는 잘 분리되고(inter-class dispersion), 클래스 내에서는 밀집되도록(intra-class compactness) 유도한다.

주요 기여 사항은 다음과 같다:

* **약한 지도(weakly-supervised) 다중 어텐션 지역화 모델 제시**: 의미론적 설명의 안내에 따라 핵심 영역을 공동으로 발견하고 특징 표현을 학습하는 Zero-shot 인식 모델을 제시한다.
* **다중 어텐션 손실(Multi-Attention Loss) 제안**: 어텐션 맵에 기하학적 제약(geometric constraints)을 적용하여 어텐션 분포가 각 맵에서 밀집되고(compact) 서로 다른 맵 간에는 다양하도록(diverse) 유도하는 손실 함수를 제안한다.
* **전역 및 지역 특징의 공동 학습**: 임베딩 소프트맥스 손실과 클래스-중심 삼중항 손실의 감독 하에 전역(global) 및 지역(local) 특징을 공동으로 학습하여 Zero-shot 인식을 위한 강화된 시각적 표현을 제공한다.
* **광범위한 실험 및 분석**: 세 가지 Zero-shot learning 데이터셋에 대한 광범위한 실험 및 분석을 통해 제안된 방법이 부분 탐지(part detection)와 Zero-shot learning 모두에서 우수한 성능을 보임을 입증한다.

## 📎 Related Works

논문은 관련 연구를 크게 두 가지 범주로 나누어 설명하고 있다: Zero-Shot Learning (ZSL) 방법론과 다중 어텐션 지역화(Multi-Attention Localization) 방법론.

### Zero-Shot Learning (ZSL) 방법론

초기 ZSL 연구들은 속성(attribute)을 중간 정보로 활용하여 이미지의 레이블을 추론했다 [22]. 그러나 현재 대부분의 ZSL 접근 방식은 문제를 시각-의미론적 임베딩(visual-semantic embedding) 문제로 취급한다.

* **임베딩 기반 방법**: ALE [8]는 랭킹 손실(ranking loss)을 사용하여 이미지 공간과 속성 공간 간의 이선형 호환성 함수(bilinear compatibility function)를 학습하며, ESZSL [23]은 릿지 회귀 손실(ridge regression loss)을 사용한다. LatEm [9]은 다중 선형 매핑 선택을 통해 조각별 선형 모델(piecewise linear model)을 학습한다. DEM [14]은 비선형 활성화 함수 ReLU를 사용하는 딥 ZSL 모델을 제시한다.
* **생성 모델 기반 방법**: GAN [16], VAE [17]와 같은 생성 모델의 성공을 활용하여, 최근 몇몇 방법론 [5, 10, 11]은 보지 못한 클래스(unseen classes)의 시각 특징을 생성(hallucinating)하여 ZSL을 일반적인 객체 인식 문제로 전환한다.
* **판별 특징 학습의 한계**: 위에서 언급된 모든 방법은 **판별적인 시각 특징 학습의 중요성을 간과**한다. CNN 모델은 전통적인 객체 인식 작업을 위해 사전 훈련되었기 때문에, 추출된 특징이 ZSL 작업에 충분히 대표적이지 않을 수 있다. 특히 미세 분류 시나리오에서는 전역 시각 특징만으로는 클래스 간의 미묘한 차이를 포착하기 어렵다.
* **기존 end-to-end 방법의 한계**: SCoRe [13] 및 LDF [19]와 같은 몇몇 최근 end-to-end 학습 방법들은 판별적 시각 정보를 발견하는 데 더 적합하지만, 여전히 전체 이미지의 전역 시각 특징만을 추출하며, 이미지 내 판별적인 부분 영역의 영향을 고려하지 않는다. 특히 LDF [19]는 배경 노이즈를 잘라내어 객체 중심 영역을 지역화하는 계단식 확대(cascaded zooming) 메커니즘을 사용하지만, 이는 여전히 전체 객체만을 발견하는 것에 국한된다.

### 다중 어텐션 지역화(Multi-Attention Localization)

* **강한 지도(strongly-supervised) 방법**: 몇몇 이전 방법들은 부분 바운딩 박스(part bounding boxes)의 추가 주석을 활용하여 미세 분류 ZSL을 위한 중요한 영역을 지역화한다 [24, 25, 5]. 예를 들어, [24]는 주석이 달린 부분 영역을 ImageNet으로 사전 훈련된 CNN에 넣어 부분 특징을 추출한다. [25, 5, 26]은 실제 주석을 사용하여 다중 부분 탐지기를 훈련하고, 기존 인식 작업을 통해 부분 특징을 학습한다. 그러나 이러한 방법들은 부분 주석에 대한 많은 인적 노동을 요구하여 대규모 실제 문제에 적용하기 어렵다.
* **약한 지도(weakly-supervised) 방법**: 따라서 ZSL 시나리오에서는 약한 지도 방식으로 부분 어텐션을 학습하는 것이 바람직하다. [27, 28]은 특정 패턴에 일관되게 반응하는 필터를 분석하여 부분 탐지기 세트를 학습한다. 공간 변형 네트워크(Spatial Transformer Networks, STN) [29]는 네트워크 내에서 데이터의 공간 조작을 명시적으로 허용하는 학습 가능한 모듈을 제안한다. [30, 28, 31]에서는 컨볼루션 채널 응답(convolutional channel responses)으로부터 후보 부분 모델을 학습한다.

### 기존 접근 방식과의 차별점

이 논문은 기존 연구의 한계를 극복하기 위해 다음과 같은 세 가지 측면에서 차별점을 가진다:

1. **의미론적 안내를 통한 부분 어텐션 학습**: 기존 어텐션 지역화 모델은 분류 손실(classification loss)의 감독 하에 부분을 발견하는 반면, 본 논문은 컨볼루션 채널 응답으로부터 부분 어텐션 모델을 학습하며, **의미론적 안내(semantic guidance)를 통해 부분을 발견**하여 ZSL에 더 판별적인 부분이 지역화되도록 한다.
2. **공동 최적화**: ZSL 모델과 어텐션 지역화 모델이 **공동으로 훈련**되어, 부분 지역화가 Zero-shot 객체 인식에 최적화되도록 보장한다. 이는 전체 모델을 end-to-end 방식으로 학습하는 것을 의미한다.
3. **다중 미세 부분 발견**: LDF [19]가 전체 객체 영역에 초점을 맞추는 것과 달리, 본 논문은 **여러 개의 더 미세한 부분 영역(multiple finer part regions)** (예: 머리, 꼬리)을 발견하여 ZSL에 더욱 판별적인 특징을 제공한다.

## 🛠️ Methodology

제안하는 Semantic-Guided Multi-Attention Localization (SGMA) 모델은 그림 1에 나타난 바와 같이 세 가지 주요 모듈로 구성된다: 다중 어텐션 서브넷 (Multi-Attention Subnet), 영역 자르기 서브넷 (Region Cropping Subnet), 그리고 공동 특징 학습 서브넷 (Joint Feature Learning Subnet). 이들은 end-to-end 방식으로 학습된다.

**문제 정의**: $C_s$ 개의 본 클래스(seen classes)에서 얻은 $N$ 개의 레이블링된 인스턴스 $D_s = \{(x_i^s, y_i^s, s_i^s)\}_{i=1}^N$를 훈련 데이터로 사용한다. 여기서 $x_i^s$는 이미지, $y_i^s \in Y_s$는 해당 클래스 레이블, $s_i^s = \phi(y_i^s) \in S$는 해당 클래스의 의미론적 표현이다. 보지 못한 클래스(unseen class)의 이미지 $x_i^u$와 보지 못한 클래스들의 의미론적 표현 $S_u = \{s_i^u = \phi(y_i^u)\}_{i=1}^{C_u}$가 주어졌을 때, ZSL의 목표는 이미지 $x_i^u$의 클래스 레이블 $y_i^u \in Y_u$를 예측하는 것이다. 여기서 $Y_s$와 $Y_u$는 서로 배타적이다.

### 전체 파이프라인 또는 시스템 구조

1. **다중 어텐션 서브넷 (Multi-Attention Subnet)**: 입력 이미지를 받아 객체의 뚜렷한 부분에 해당하는 여러 개의 어텐션 맵 $M_i$를 생성한다.
2. **영역 자르기 서브넷 (Region Cropping Subnet)**: 생성된 어텐션 맵을 기반으로 미분 가능한(differentiable) 연산을 통해 판별적인 부분 영역을 자른다.
3. **공동 특징 학습 서브넷 (Joint Feature Learning Subnet)**: 잘라낸 부분 이미지들과 원본 이미지를 입력으로 받아 전역 및 지역 시각 특징을 학습하고, 최종 ZSL 인식을 수행한다.

**Figure 1**: 제안하는 Semantic-Guided Multi-Attention localization model (SGMA)의 프레임워크. 모델은 원본 이미지를 입력으로 받아 $n$개의 부분 어텐션 맵($n=2$)을 생성한다. Multi-Attention loss $L_{MA}$는 각 맵의 어텐션 영역을 밀집하게 유지하고 서로 다른 맵 간에는 다양하게 만든다. Cropping subnet에서 얻은 부분 이미지와 원본 이미지는 semantic description-guided 객체 인식을 위해 Joint Feature Learning Subnet의 다른 CNN에 입력된다.

### 각 주요 구성 요소 및 역할

#### 3.1 다중 어텐션 서브넷 (Multi-Attention Subnet)

이 서브넷은 입력 이미지에서 여러 개의 어텐션 맵을 생성하여 객체의 미세한 판별 영역을 포착한다.

**어텐션 맵 생성 과정**:

1. **CNN 백본**: 입력 이미지는 먼저 컨볼루션 네트워크 백본을 통과하여 크기 $H \times W \times C$의 특징 표현을 얻는다.
2. **채널 설명 벡터 (Channel Descriptor)**: 추출된 특징 맵에 전역 평균 풀링(Global Average Pooling)을 적용하여 공간 정보를 인코딩하는 채널 설명 벡터 $p$를 얻는다. $c$-번째 요소 $p_c$는 다음과 같이 계산된다:
    $$ p_c = \frac{1}{H \times W} \sum_{i=1}^H \sum_{j=1}^W b_c(i,j) $$
    여기서 $b_c(i,j)$는 $c$-번째 채널의 특징 맵에 있는 $(i,j)$ 위치의 값이다.
3. **채널별 어텐션 가중치 (Channel-wise Attention Weight)**: 채널 설명 벡터 $p$에 스택된 완전 연결(FC) 레이어와 ReLU 활성화 함수 $f(\cdot)$를 적용한 후, 시그모이드(sigmoid) 활성화 함수 $\sigma(\cdot)$를 통해 채널별 어텐션 가중치 $a_i$를 얻는다. 이는 $i$-번째 부분과 관련된 채널의 소프트 어텐션 가중치로 해석된다.
    $$ a_i = \sigma(W_2 f(W_1 p)) $$
    여기서 $W_1, W_2$는 학습 가능한 가중치 행렬이다.
4. **어텐션 맵 생성**: $i$-번째 부분에 대한 어텐션 맵 $M_i(x)$는 모든 채널의 가중치 합과 시그모이드 활성화 함수를 통해 생성된다.
    $$ M_i(x) = \sigma \left( \sum_{c=1}^C a_i^c f_{Conv}(x)^c \right) $$
    여기서 $a_i^c$는 $i$-번째 부분에 대한 $c$-번째 채널의 어텐션 가중치이며, $f_{Conv}(x)^c$는 CNN 백본에서 추출된 $c$-번째 채널의 특징 맵이다. 시그모이드 활성화는 어텐션 맵이 네트워크가 판별적인 부분에 집중하도록 게이팅 메커니즘(gating mechanism) 역할을 한다.

**다중 어텐션 손실 (Multi-Attention Loss, $L_{MA}$)**
여러 어텐션 맵이 서로 다른 판별 영역을 발견하도록 장려하기 위해 기하학적 제약(geometric constraints)을 적용하여 다중 어텐션 손실을 설계한다. 이 손실은 **밀집도 손실 (Compactness Loss, $L_{CPT}$)**과 **다양성 손실 (Diversity Loss, $L_{DIV}$)** 두 가지 구성 요소로 이루어져 있다.
$$ L_{MA} = \sum_i^{N_a} [L_{CPT}(M_i) + \lambda L_{DIV}(M_i)] $$
여기서 $N_a$는 어텐션 맵의 수이며, $\lambda$는 균형 요소이다.

* **밀집도 손실 (Compactness Loss, $L_{CPT}(M_i)$)**:
    어텐션 맵이 분산되지 않고 최고 활성화 지점 주위에 집중되도록 유도한다. $i$-번째 어텐션 맵의 최고 활성화 지점(peak activation)에 가우시안 블롭(Gaussian blob) 형태의 이상적인 집중 어텐션 맵 $\tilde{M}_i$를 생성하고, 실제 생성된 어텐션 맵 $M_i$와의 $L_2$ 거리를 최소화한다.
    $$ L_{CPT}(M_i) = \frac{1}{|Z|} \sum_{z \in Z} ||m_z^i - \tilde{m}_z^i||_2^2 $$
    여기서 $m_z^i$와 $\tilde{m}_z^i$는 각각 $z$ 위치에서의 생성된 어텐션 맵과 이상적인 집중 어텐션 맵을 나타내며, $|Z|$는 어텐션 맵의 크기이다.

* **다양성 손실 (Diversity Loss, $L_{DIV}(M_i)$)**:
    어텐션 맵들이 서로 다른 판별적인 부분에 집중하도록 장려한다. 예를 들어, 하나의 맵은 머리에 집중하고 다른 맵은 꼬리에 집중하는 식이다.
    $$ L_{DIV}(M_i) = \sum_{z \in Z} m_z^i \max\{0, \hat{m}_z - mrg\} $$
    여기서 $\hat{m}_z = \max_{k \neq i} m_z^k$는 $z$ 위치에서 다른 어텐션 맵들 중 최대 활성화 값을 나타내며, $mrg$는 마진 값이다. 이 최대-마진(maximum-margin) 설계는 손실이 노이즈에 덜 민감하도록 하여 강건성을 향상시킨다. 하나의 어텐션 맵의 특정 위치 활성화가 높을 때, 이 손실은 동일한 위치에서 다른 어텐션 맵의 활성화를 낮게 유지하도록 유도한다.

#### 3.2 영역 자르기 서브넷 (Region Cropping Subnet)

생성된 어텐션 맵을 활용하여 객체의 부분을 자른다. 이 자르기 연산은 역전파(backward propagation)를 통해 최적화될 수 있도록 미분 가능하게 설계된다.

1. **파라미터 예측**: 자르기 네트워크 $f_{CNet}(\cdot)$는 어텐션 맵 $M_i$를 입력으로 받아 세 가지 파라미터 $[t_x, t_y, t_s]$를 출력한다.
    $$ [t_x, t_y, t_s] = f_{CNet}(M_i) $$
    여기서 $t_x, t_y$는 사각형 중심의 x, y 좌표이고, $t_s$는 사각형의 변 길이이다. $f_{CNet}$은 두 개의 완전 연결(FC) 레이어로 구성된다.
2. **연속 마스크 생성**: 두 차원 연속 박스카 마스크(two-dim continuous boxcar mask) $V(x,y) = V_x \cdot V_y$를 생성한다.
    $$ V_x = f(x - t_x + 0.5t_s) - f(x - t_x - 0.5t_s) $$
    $$ V_y = f(y - t_y + 0.5t_s) - f(y - t_y - 0.5t_s) $$
    여기서 $f(x) = 1/(1 + \exp(-kx))$는 시그모이드와 유사한 형태의 함수로, $k$는 기울기를 조절하는 하이퍼파라미터이다.
3. **영역 자르기 및 리사이즈**: 잘라낸 영역 $x_{part}^i$는 원본 이미지 $x$와 연속 마스크 $V_i$의 요소별 곱셈으로 얻어진다: $x_{part}^i = x \odot V_i$. 이어서 빌리니어 보간법(bilinear interpolation)을 사용하여 잘라낸 영역을 원본 이미지와 동일한 크기로 리사이즈한다.

#### 3.3 공동 특징 학습 서브넷 (Joint Feature Learning Subnet)

ZSL을 위한 강화된 시각적 표현을 제공하기 위해 원본 이미지와 부분 이미지(region cropping subnet에서 생성된)로부터 전역 및 지역 시각 특징을 공동으로 학습한다.

* **CNN 백본**: 원본 이미지와 각 부분 이미지는 224x224 크기로 리사이즈된 후 개별 CNN 백본 네트워크(동일한 VGG19 아키텍처)에 입력된다. 컨볼루션 레이어 다음에는 전역 평균 풀링(Global Average Pooling)이 이어져 시각 특징 벡터 $\theta(x)$를 얻는다.
* **공동 손실 함수**: ZSL 작업에 대한 판별적인 특징을 학습하기 위해 두 가지 협력적인 손실을 사용한다: **임베딩 소프트맥스 손실 ($L_{CLS}$)**과 **클래스-중심 삼중항 손실 ($L_{CCT}$)**. $L_{CLS}$는 클래스 간의 구별력을 높이고, $L_{CCT}$는 각 클래스의 학습된 특징이 클래스 내에서 집중되도록 하여 클래스 내 발산(intra-class divergence)을 낮춘다.

**임베딩 소프트맥스 손실 (Embedding Softmax Loss, $L_{CLS}$)**:
의미 특징 $\phi(y)$에 대해, 다중 모델 특징의 호환성 점수(compatibility score)는 $s = \theta(x)^T W \phi(y)$로 정의된다. 여기서 $W$는 학습 가능한 변환 행렬이다. 이 호환성 점수를 소프트맥스 로짓(logits)으로 사용하여 임베딩 소프트맥스 손실은 다음과 같이 주어진다:
$$ L_{CLS} = -\frac{1}{N} \log \frac{\exp(s_j)}{\sum_{y' \in Y_s} \exp(s_{y'})} $$
여기서 $s_j = \theta(x)^T W \phi(y_j)$이며, $y_j \in Y_s$는 본 클래스 레이블, $N$은 훈련 샘플 수이다.
전역 및 지역 특징을 모델의 복잡성을 증가시키지 않고 결합하기 위해 **지연 융합 전략 (late fusion strategy)**을 채택한다. 전체 호환성 점수는 각 CNN에서 얻은 호환성 점수들을 합산하여 계산되며, 이를 소프트맥스 손실 계산에 사용한다. 구체적으로, 방정식 (9)의 $s_j$는 $\sum_i (s_j^i)$로 대체되며, 여기서 $s_j^i = \theta_i(x)^T W_i \phi(y_j)$이고 $i$는 부분 이미지 및 원본 이미지의 인덱스이다.

**클래스-중심 삼중항 손실 (Class-Center Triplet Loss, $L_{CCT}$)**:
이 손실은 원래 얼굴 인식 작업에서 딥 시각 특징의 클래스 내 거리를 최소화하기 위해 설계되었다 [21]. 이 논문에서는 특징의 클래스 내 밀집도를 장려하기 위해 네트워크를 클래스-중심 삼중항 손실과 함께 훈련한다. 클래스 인덱스를 $i, k$라고 할 때, 손실은 다음과 같이 공식화된다:
$$ L_{CCT} = \max\{0, mrg + ||\hat{\varphi}_i - \hat{C}_i||_2^2 - ||\hat{\varphi}_i - \hat{C}_k||_2^2\}_{i \neq k} $$
여기서 $mrg$는 마진, $\varphi_i = \theta(x)^T W_i$는 의미 특징 공간으로 매핑된 시각 특징이며, $C_i$는 각 클래스의 학습 가능한 "중심" 파라미터이다. $\hat{\cdot}$는 $L_2$ 정규화 연산을 의미한다. 정규화 연산은 특징점들을 단위 초구(unit hypersphere) 표면에 위치시켜 적절한 마진 설정이 용이하도록 한다. 이 손실은 일반 삼중항 손실에서 요구되는 삼중항 샘플링의 필요성을 제거한다.

**전체 목표 함수 (Overall Objective Function)**:
제안하는 SGMA 모델은 다음 목적 함수로 end-to-end 방식으로 훈련된다:
$$ L_{SGMA} = L_{MA} + \alpha_1 L_{CLS} + \alpha_2 L_{CCT} $$
여기서 균형 요소 $\alpha_1$과 $\alpha_2$는 모든 실험에서 1로 설정된다.

### 3.4 SGMA 모델로부터 추론 (Inference from SGMA Model)

SGMA 모델로부터 보지 못한 클래스 이미지의 레이블을 추론하는 두 가지 방법이 있다.

1. **최대 호환성 점수 기반 추론 (Embedding Softmax Branch)**:
    가장 직접적인 방법으로, 그림 1의 녹색 경로와 같이 최대 전체 호환성 점수를 가진 클래스 레이블을 선택한다. $s_y = \theta(x)^T W \phi(y)$이다.

2. **클래스-중심 특징 기반 추론 (Class-Center Triplet Branch)**:
    클래스-중심 브랜치에서 학습된 특징 $\varphi_{cct}(x)$를 활용한다. 이는 그림 1의 보라색 경로이다. 테스트 이미지의 특징 $\varphi_{cct}(x)$와 보지 못한 클래스의 프로토타입 $\Phi_u^{cct}$ 간의 유사성을 기반으로 클래스 레이블을 추론할 수 있다.
    * **보지 못한 클래스 프로토타입 생성**: 보지 못한 클래스의 의미론적 설명이 본 클래스의 선형 조합으로 표현될 수 있다고 가정한다. 이러한 조합의 가중치 행렬 $W$는 릿지 회귀(ridge regression)를 풀어 얻을 수 있다:
        $$ W = \arg \min_W ||\Phi_u - W\Phi_s||_2^2 + \lambda ||W||_2^2 $$
        여기서 $\Phi_u$와 $\Phi_s$는 각각 보지 못한 클래스와 본 클래스의 의미론적 행렬이며, 각 행은 클래스의 의미 벡터이다.
    * **프로토타입 적용**: 학습된 $W$를 사용하여 보지 못한 클래스의 프로토타입을 얻을 수 있다: $\Phi_u^{cct} = W\Phi_s^{cct}$. 여기서 $\Phi_s^{cct}$는 각 클래스의 모든 이미지 특징을 평균하여 얻은 본 클래스의 프로토타입이다.
    * **유사성 기반 예측**: 클래스 레이블은 테스트 이미지의 특징 $\varphi_{cct}(x)$와 $\Phi_u^{cct}$의 각 클래스 프로토타입 간의 내적 유사성에 기반하여 추론된다.

3. **두 추론 방법의 결합**:
    전역 및 지역 이미지 설명을 결합하기 위해 서로 다른 CNN에서 생성된 시각 특징을 연결(concatenate)한다. 또한, 위 두 가지 추론 방법의 결과를 결합하기 위해, 임베딩 소프트맥스 브랜치의 호환성 점수 $s_y$와 클래스-중심 삼중항 브랜치의 유사성 점수를 더하여 테스트 이미지에 대한 최종 예측 점수를 얻는다:
    $$ y = \arg \min_{y \in Y_U} (s_y + \beta \langle \varphi_{cct}(x), [\Phi_u^{cct}]_y \rangle) $$
    여기서 $\langle \cdot \rangle$는 내적을 의미하고, $[\cdot]_y$는 클래스 $y$에 해당하는 행을 나타내며, $\beta$는 클래스-중심 브랜치의 기여도를 조절하는 균형 요소이다.

  * 공동 특징 학습 서브넷 사전 훈련: 사전 훈련된 자르기 서브넷에서 얻은 잘라낸 부분을 사용하여 공동 특징 학습 서브넷을 사전 훈련한다.

이러한 결과는 탐지된 부분이 객체의 의미론적 부분과 잘 일치함을 보여준다.

* **모델 복잡성**: 이미지와 부분 패치에 동일한 CNN을 공유 가중치로 사용할 경우 모델 복잡성은 감소하지만, ZSL 성능은 약간 저하된다 (CUB-PS 및 AWA-PS에서 각각 3.6%, 2.7% 감소).

### 4.4 일반화된 Zero-Shot Learning (Generalized Zero-Shot Learning) 결과

테스트 이미지가 본 클래스와 보지 못한 클래스 모두에서 나오는 일반화된 ZSL 설정에서도 방법을 평가한다. 본 클래스와 보지 못한 클래스에서 테스트 이미지를 분류하는 성능은 각각 $A_U$와 $A_S$로 보고되며, 조화 평균(harmonic mean) $H = \frac{2 \cdot A_S \cdot A_U}{A_S + A_U}$로 측정된다.

| Method    | CUB ($A_U$) | CUB ($A_S$) | CUB (H)  | AwA ($A_U$) | AwA ($A_S$) | AwA (H)  |
|:----------|:------------|:------------|:---------|:------------|:------------|:---------|
| DEM [14]  | 19.6        | 57.9        | 29.2     | 32.8        | 84.7        | 47.3     |
| GAZSL [5] | 31.7        | 61.3        | 41.8     | 29.6        | 84.2        | 43.8     |
| LDF [19]  | 26.4        | 81.6        | 39.9     | 9.8         | 87.4        | 17.6     |
| Ours      | **36.7**    | **71.3**    | **48.5** | **37.6**    | **87.1**    | **52.5** |

**분석**:

* 본 모델은 조화 평균 $H$ 점수에서 이전 최첨단 방법론들을 능가한다.
* 특히 미세 분류 클래스 간의 미묘한 차이를 포착하는 데 판별적인 부분이 중요한 CUB 데이터셋에서 $H$ 점수를 6.7% 향상시켰다 (48.5% 대 41.8%).

### 4.5 Ablation Study (구성 요소별 분석)

기준선은 부분 지역화 없이 임베딩 소프트맥스 손실만 사용하는 모델이다.

| Method                | CUB      | AWA      | FLO      | Avg      |
|:----------------------|:---------|:---------|:---------|:---------|
| Baseline              | 60.2     | 61.5     | 57.7     | 59.8     |
| Parts                 | 55.4     | 51.2     | 49.8     | 52.1     |
| Baseline+Parts        | **67.4** | **64.3** | **63.9** | **65.2** |
| Baseline+Random Parts | 56.3     | 59.8     | 56.4     | 57.5     |
| Embedding Softmax     | 60.9     | 62.4     | 57.2     | 60.2     |
| Class-Center Triplet  | 62.1     | 64.6     | 61.1     | 62.6     |
| Combined              | **63.5** | **65.7** | **61.8** | **63.7** |

**판별적인 영역의 효과**:

* 부분 영역만 사용하는 모델("Parts")은 가장 낮은 성능을 보인다. 이는 부분 특징이 국소적인 정보만 제공하며 전체 객체를 잘 인식하지 못하기 때문이다.
* 원본 이미지와 지역화된 부분을 결합한 모델("Baseline+Parts")은 기준선보다 5.4% (65.2% 대 59.8%) 크게 성능이 향상된다.
* 원본 이미지에 무작위로 자른 부분("Baseline+Random Parts")을 추가하면 대부분의 경우 성능이 저하된다. 이는 무작위 부분이 의미론적 정렬(semantic alignment)이 부족하기 때문으로 분석된다 (예: 한 이미지에서는 머리이지만 다른 이미지에서는 다리가 될 수 있음). 이는 본 모델의 지역화된 부분이 더 나은 의미론적 정렬을 제공함을 뒷받침한다.

**공동 손실의 효과**:

* 공동 손실로 훈련하고 원본 이미지만 입력으로 사용하는 모델의 추론 결과를 보여준다.
* 임베딩 소프트맥스 브랜치에서 추론된 결과는 클래스-중심 삼중항 손실이 판별 특징을 강화하는 정규화자 역할을 할 수 있어 기준선보다 약간 개선된다 (60.2% 대 59.8%).
* 클래스-중심 삼중항 브랜치에서 추론된 결과는 더 좋고, 이 두 브랜치의 추론을 결합했을 때("Combined") 가장 좋은 결과를 얻는다. 이는 기준선 결과보다 3.9% 향상된 성능이다 (63.7% 대 59.8%).

## 🧠 Insights & Discussion

**논문에서 뒷받침되는 강점**:

* **판별적 부분 특징의 중요성 입증**: 본 연구는 Zero-shot learning에서 객체의 판별적 부분(discriminative parts)을 식별하고 활용하는 것이 얼마나 중요한지를 명확하게 보여준다. 특히 미세 분류 데이터셋에서 성능이 크게 향상된 점은 이 주장을 강력하게 뒷받침한다.
* **약한 지도 어텐션 지역화의 성공**: 사람의 수동 주석 없이 의미론적 설명의 안내를 받아 객체의 핵심 부분을 자동으로 발견하는 약한 지도(weakly-supervised) 방식의 성공은 실제 대규모 ZSL 시스템 구축에 큰 장점을 제공한다. 이는 기존의 수동 주석에 대한 높은 비용 문제를 해결한다.
* **다중 어텐션 손실의 효과**: 다중 어텐션 손실 ($L_{MA}$)은 각 어텐션 맵이 특정 영역에 밀집되고 서로 다른 맵들이 다양한 부분을 포착하도록 유도하여 모델의 부분 탐지 능력을 향상시키는 데 결정적인 역할을 한다. ablation study에서 $L_{MA}$를 제거했을 때 성능이 크게 저하된 점으로 확인된다.
* **전역 및 지역 특징의 시너지**: 전체 이미지의 전역 특징과 발견된 부분의 지역 특징을 함께 학습함으로써, 더욱 풍부하고 판별적인 시각적 표현을 얻을 수 있음을 입증한다. 이는 다양한 스케일의 정보를 통합하는 효과적인 전략이다.
* **협력적 손실 함수의 효과**: 임베딩 소프트맥스 손실과 클래스-중심 삼중항 손실을 공동으로 사용하여 클래스 간 분리도와 클래스 내 밀집도를 동시에 강화하는 전략은 학습된 특징의 판별력을 크게 향상시킨다. 이는 ablation study에서 두 손실을 결합했을 때 가장 좋은 결과를 보인 점에서 확인된다.
* **End-to-end 학습의 효율성**: 전체 파이프라인을 end-to-end 방식으로 학습함으로써, 각 구성 요소가 ZSL 작업에 최적화되도록 하여 기존의 비 end-to-end 방식보다 일관되게 우수한 성능을 달성한다.

**한계, 가정 또는 미해결 질문**:

* **부분 주석 부재로 인한 정밀도 한계**: 부분 탐지 결과에서 SPDA-CNN(부분 주석 사용)과 비교할 때 여전히 성능 차이가 존재한다(61.5% 대 79.1%). 이는 약한 지도 방식의 내재적인 한계로, 정밀한 부분 주석 없이 최고 수준의 정밀도에 도달하기 어려울 수 있음을 시사한다.
* **부분 개수(Number of Parts) 설정**: 논문에서는 부분의 개수를 2개로 설정했는데, 이는 더 많은 맵이 중복을 유발하고 성능 향상에 기여하지 않기 때문이라고 언급한다. 이는 모델이 복잡한 객체에 대해 더 많은 미세한 부분을 효과적으로 탐지하거나 통합하는 데 제한이 있을 수 있음을 암시한다. 더 많은 부분을 유연하게 다루는 메커니즘이 필요할 수 있다.
* **Square Shape 가정**: 영역 자르기 서브넷에서 부분 영역의 모양을 계산 효율성을 위해 사각형으로 가정한다. 실제 객체 부분은 다양한 비정형 모양을 가질 수 있으므로, 이러한 가정은 부분의 정확한 경계를 포착하는 데 한계가 될 수 있다.
* **하이퍼파라미터 튜닝의 영향**: $mrg$, $k$, $\lambda$, $\alpha_1$, $\alpha_2$와 같은 많은 하이퍼파라미터가 그리드 서치를 통해 결정된다. 이러한 하이퍼파라미터에 대한 민감도 분석이 더 광범위하게 이루어지지 않아, 다른 데이터셋이나 시나리오에 대한 일반화 가능성에 대한 의문을 남길 수 있다.
* **의미론적 표현의 품질**: 논문은 [40, 10]에서 제공하는 속성 또는 문장 특징을 의미론적 표현으로 사용한다. 이러한 의미론적 표현의 품질이 전체 ZSL 성능에 큰 영향을 미칠 수 있으나, 본 연구에서는 의미론적 특징 자체의 개선보다는 이를 활용하는 방법에 초점을 맞춘다.

**논문에 근거한 간략한 비판적 해석 및 논의사항**:

이 논문은 ZSL의 핵심 과제인 판별적인 시각 특징 학습의 중요성을 효과적으로 다루고, 특히 미세 분류 시나리오에서 혁신적인 해결책을 제시한다. 다중 어텐션 메커니즘과 새로운 손실 함수를 통해 약한 지도 환경에서도 객체의 의미론적 부분들을 성공적으로 지역화하고 활용한 점은 매우 고무적이다. end-to-end 학습과 공동 손실 함수의 사용은 특징의 판별력을 극대화하여 기존 방법론들을 뛰어넘는 성능을 달성했다.

그러나 부분 탐지의 정밀도를 더욱 높이거나, 다양한 모양의 부분을 더욱 유연하게 처리할 수 있는 방법을 탐구하는 것은 미래 연구의 방향이 될 수 있다. 또한, 부분 개수의 자동 결정 메커니즘이나 다른 의미론적 표현 방식과의 결합 가능성도 논의해 볼 가치가 있다. 전반적으로 이 연구는 Zero-shot learning 분야에서 시각 특징의 '어디를 볼 것인가'에 대한 중요한 질문에 답하며, 더욱 강건하고 설명 가능한 ZSL 모델 개발의 길을 열었다고 평가할 수 있다.

## 📌 TL;DR

이 논문은 Zero-shot learning (ZSL)에서 판별적인 시각 특징, 특히 객체의 핵심 부분(discriminative parts) 학습의 중요성을 강조하며, **Semantic-Guided Multi-Attention Localization (SGMA)** 모델을 제안한다. SGMA는 사람의 주석 없이 의미론적 표현의 안내를 받아 객체의 여러 판별적 부분을 자동으로 탐지한다. 이를 위해 **다중 어텐션 서브넷**이 **다중 어텐션 손실 ($L_{MA}$, 밀집도 $L_{CPT}$와 다양성 $L_{DIV}$ 포함)**을 통해 부분 어텐션 맵을 생성하고, **영역 자르기 서브넷**은 미분 가능한 방식으로 해당 부분을 잘라낸다. 마지막으로 **공동 특징 학습 서브넷**은 원본 이미지의 전역 특징과 잘라낸 부분의 지역 특징을 **임베딩 소프트맥스 손실 ($L_{CLS}$)**과 **클래스-중심 삼중항 손실 ($L_{CCT}$)**의 공동 감독 하에 end-to-end 방식으로 학습하여 클래스 간 분리도와 클래스 내 밀집도를 동시에 최적화한다.

광범위한 실험을 통해 SGMA는 세 가지 ZSL 벤치마크 데이터셋에서 최첨단 성능을 크게 능가하며, 특히 미세 분류 작업에서 강력한 효과를 입증한다. 이 연구는 약한 지도 부분 탐지의 효과와 전역 및 지역 특징의 시너지를 통해, 기존 ZSL 방법론들이 간과했던 시각 특징의 판별력 문제를 효과적으로 해결하여 실제 적용 및 향후 연구에 중요한 기여를 할 가능성을 보여준다.
