# Medical Image Segmentation

## 서론

### 1. 연구 배경

의료 영상 분할 (Medical Image Segmentation) 분야는 최근 CNN 기반의 계층적 U-Net 아키텍처에서 Transformer 기반 전역 컨텍스트 모델링으로 진화하고 있다. 2018 년부터 2024 년까지 발행된 주요 논문들을 분석해본 결과, 의료 영상 (CT, MRI, 2D/3D 스캔) 에 대한 픽셀/복셀 단위 분할 마스크 생성을 위한 다양한 방법론들이 발전해 왔음을 확인하였다.

### 2. 문제의식 및 분석 필요성

의료 영상 분할 연구는 학습/주석 방식, 최적화/데이터 전략, 모델 아키텍처, 문제 정의 관점이라는 4 차원의 기준에 따라 체계적으로 분류될 필요가 있다. 특히 SAM 이라는 파운데이션 모델 등장 이후 제로샷 및 인터랙티브 접근이 활발해지고 있으며, CNN 과 Transformer 의 장점을 결합한 하이브리드 아키텍처 연구가 핵심 흐름으로 부상하고 있어 이러한 흐름을 체계적으로 정리하는 것이 중요해졌다.

### 3. 보고서의 분석 관점

본 보고서는 다음 세 가지 관점에서 문헌을 정리하고 분석한다:

1. **연구체계 분류**: 학습/주석 방식 (지도학습, 약지도, 반지도, 자기 지도, 제로샷), 최적화 전략, 모델 아키텍처, 문제 정의 관점에 따라 연구들을 체계적으로 분류
2. **방법론 분석**: 테스트-시간 적응, 자동 최적화, Transformer 기반, 약지도 학습, 베이시안 모델링 등 방법론 계열별 공통 문제 설정 및 접근 구조 비교 분석
3. **실험결과 분석**: 주요 데이터셋별 성능, 평가 지표, 도메인 일반화 능력, 계산 효율성 등 실험 결과를 정렬하고 성능 패턴 및 경향 분석

### 4. 보고서 구성

1 장에서는 학습/주석 방식, 모델 아키텍처, 최적화 전략, 문제 정의 관점이라는 4 차원의 기준에 따라 의료 영상 분할 연구들을 체계적으로 분류하며, 각 분야별 주요 논문들을 표로 정리하고 연구 기여도를 분석한다.

### 3. 종합 정리

의료 영상 분할 연구는 4 가지 주요 축으로 정리됨. 첫째, 학습/주석 방식으로 지도학습, 약지도/반지도 학습, 자기/제로샷 학습, 도메인 일반화로 구분되며, 둘째, 최적화/데이터 전략으로 자동 데이터 증강, 평가 지표 가이드라인 등이 있으며, 셋째, 모델 아키텍처 관점에서 CNN, Transformer, 하이브리드, 그래프 합성곱 등이 적용되고, 넷째, 문제 정의 관점에서 도메인 일반화, 프롬프트 기반/인터랙티브, 3D/멀티모달리티 적용 등이 확인됨. 특히 SAM 이라는 파운데이션 모델 등장 이후 제로샷 및 인터랙티브 접근이 활발해지고 있으며, ConvNet 과 Transformer 의 장점을 결합한 하이브리드 아키텍처 연구가 핵심 흐름으로 부상하고 있음.

## 2 장. 방법론 분석

### 1. 공통 문제 설정 및 접근 구조

의료 영상 분할(MIS) 분야의 방법론들은 공통적으로 **입력 영상 → 처리 파이프라인 → 분할 출력**의 3 단계 구조를 따르며, 다음과 같은 공통 특성을 공유한다.

| 공통 요소       | 상세 내용                                                |
| --------------- | -------------------------------------------------------- |
| **입력**        | 의료 영상 (CT, MRI, 2D/3D 스캔)                          |
| **출력**        | 픽셀/복셀 단위 분할 마스크 ($Z \in \{0,1\}^{ \| X \| }$) |
| **구조적 핵심** | 인코더-디코더 아키텍처                                   |
| **손실 함수**   | Dice loss, Cross-entropy, 또는 혼합형                    |

### 2. 방법론 계열 분류

#### 2.1 테스트-시간 적응 (Test-Time Adaptation, TTA) 계열

**계열 정의:** 학습 시에는 일반화되지 않은 도메인 시프트를 테스트 시점에 개별적으로 적응하는 전략

**공통 특징:**

- NI2S (Normality-to-Segmentation) 네트워크: 도메인 불변성 확보
- I2NI (Inverse-to-Normality) 네트워크: 테스트 이미지당 파라미터 적응
- DAE 기반 타당성 판정: 분할 결과의 일관성 유도

**주요 논문:**

| 방법론 계열 | 논문명                                     | 핵심 특징                                |
| ----------- | ------------------------------------------ | ---------------------------------------- |
| TTA         | Test-Time Adaptable Neural Networks (2020) | I2NI/NI2S 서브네트워크 + DAE 기반 타당성 |

**방법론 전제:** 테스트 이미지당 라벨 정보 없이 단일 샘플만으로 적응, 대규모 도메인 시프트 시 아틀라스 초기화 사용

#### 2.2 자동 최적화 및 강화 학습 계열

**계열 정의:** 데이터 증강, 아키텍처 설계 등 하이퍼파라미터를 자동 탐색하여 성능 극대화

**공통 특징:**

- 확률적 최적화 (Stochastic Optimization)
- 자연 그래디언트 기반 정책 업데이트
- 검증 손실로 자동 탐색

**주요 논문:**

| 방법론 계열 | 논문명                                           | 핵심 특징                                            |
| ----------- | ------------------------------------------------ | ---------------------------------------------------- |
| 자동 최적화 | Automatic Data Augmentation (2020)               | 7 연산 매개변수 구간 탐색 + 자연 그래디언트          |
| 자동 최적화 | Automatically Designing CNN Architectures (2018) | 정책 경사 강화 학습으로 CNN 하이퍼파라미터 직접 학습 |

**방법론 전제:** 11 차원 검색 공간, 확률적 완화 ($T=2$), 768 GPU 시간 대비 100 GPU 시간 미만

#### 2.3 Transformer/ViT 기반 파운데이션 모델 계열

**계열 정의:** 패치 임베딩과 Self-Attention을 통해 전역 컨텍스트를 모델링하는 트랜스포머 아키텍처

**공통 특징:**

- 패치 임베딩 ($N \times D$)
- Multi-Head Self-Attention ($O(N^2)$ 계산)
- 이미지넷 사전 학습 후 미세 조정

**주요 논문:**

| 방법론 계열        | 논문명                                 | 핵심 특징                                      |
| ------------------ | -------------------------------------- | ---------------------------------------------- |
| ViT                | Transformers in Medical Imaging (2022) | 패치 임베딩 + ViT 인코더 + 다운스트림 태스크   |
| 자기사전훈련       | Self Pre-training with MAE (2022)      | 75% 패치 마스킹 + 재구성 손실로 자기 사전 훈련 |
| 커널 네트워크 확장 | MedNeXt (2024)                         | ConvNeXt 완전 3D 인코더-디코더 구조 활용       |

**방법론 전제:** 고해상도 3D 볼륨 처리 시 $O(N^2)$ 복잡도 제약, 대규모 레이블링 데이터 필요

#### 2.4 약지도 및 세미-지도 학습 계열

**계열 정의:** 픽셀 단위 주석이 부족하거나 희소인 상황에서 스크리블, 가시 부분 등을 활용한 학습

**공통 특징:**

- 듀얼 브랜치 네트워크 (공유 인코더 + 독립 디코더)
- 동적 혼합 의사 레이블 (Dynamic Mixed Pseudo Labels)
- 부분 교차 엔트로피 손실

**주요 논문:**

| 방법론 계열 | 논문명                     | 핵심 특징                                           |
| ----------- | -------------------------- | --------------------------------------------------- |
| 약지도      | Scribble-Supervised (2022) | 공유 인코더 + 2 독립 디코더 + 동적 혼합 의사 레이블 |
| 클래스 중심 | ACTION++ (2023)            | 오프라인 클래스 중심 사전 계산 + 동적 온도 스케줄링 |

**방법론 전제:** 스크리블 주석이 픽셀 단위 주석과 유사한 비용 가정, 긴 꼬리 분포에서 균형 잡힌 특징 공간 학습 필요

#### 2.5 베이시안 및 통계 모델링 계열

**계열 정의:** 확률적 그래픽 모델과 베이지안 추론을 통해 해석 가능한 분할 및 일반화 달성

**공통 특징:**

- 변이 베이지안 추론 (Variational Inference)
- 계층적 베이지안 사전
- 형상/외형 잠재 변수 분해

**주요 논문:**

| 방법론 계열 | 논문명         | 핵심 특징                                                       |
| ----------- | -------------- | --------------------------------------------------------------- |
| 베이시안    | BayeSeg (2023) | SAR 공간 상관 모델 + 변이 베이지안 추론 + 도메인-안정 형상 추출 |

**방법론 전제:** 이미지 강도 값을 공간적 독립 정보와 상관 정보의 합으로 모델링, 베이지안 사전과 공분산 행렬 가정

#### 2.6 제로샷 및 파운데이션 모델 계열

**계열 정의:** 자연 영상으로 학습된 대형 모델을 의료 영상에 제로샷 또는 적소학습으로 적용

**공통 특징:**

- SAM 아키텍처 (이미지/프롬프트 인코더 + 마스크 디코더)
- 시각적 프롬프트 (점, 경계 상자)
- PEFT 어댑터로 최소 파라미터 업데이트

**주요 논문:**

| 방법론 계열 | 논문명                                                       | 핵심 특징                                                 |
| ----------- | ------------------------------------------------------------ | --------------------------------------------------------- |
| 제로샷      | SAM.MD (2023)                                                | 사전학습된 SAM 블랙박스 활용 + 점/상자 프롬프트 기반 분할 |
| 제로샷      | When SAM Meets Medical Images (2023)                         | ViT-B/L/H 변형 + 다상 통합 모듈                           |
| 도메인 적응 | Medical SAM Adapter (2023)                                   | PEFT 어댑터 + SD-Trans + HyP-Adpt로 도메인 적응           |
| 종합 서베이 | Segment Anything Model for Medical Image Segmentation (2024) | 파인튜닝/자동 프롬프팅/프레임워크 수정/3D 처리 분류       |

**방법론 전제:** 자연영상과 의료영상의 도메인 격차, 대규모 의료 영상-마스크 쌍 데이터셋 필요

#### 2.7 그래프 합성곱 계열

**계열 정의:** 그래프 구조를 활용한 전역 및 지역 정보 통합

**공통 특징:**

- KNN 기반 그래프 구성 ($K=11$ 이웃)
- Max-Relative 그래프 합성곱
- 가산 예측 통합

**주요 논문:**

| 방법론 계열   | 논문명           | 핵심 특징                                        |
| ------------- | ---------------- | ------------------------------------------------ |
| 그래프 합성곱 | G-CASCADE (2023) | GCAM(SPA-GCB) + UCB 블록 + 4 예측 헤드 가산 통합 |

**방법론 전제:** PVTv2/MERIT 계층적 트랜스포머 인코더 사용, 그래프 합성곱 블록 계산 효율성 활용

#### 2.8 채널-공간 어텐션 계열

**계열 정의:** 채널별 동적 공간 어텐션 맵 생성으로 CNN의 수용장 한계 극복

**공통 특징:**

- 채널 어텐션 (MLP 기반 평균/최대 풀링)
- 공간 어텐션 (멀티스케일 depth-wise convolution)
- 슬라이딩 윈도우 예측

**주요 논문:**

| 방법론 계열      | 논문명                    | 핵심 특징                                                        |
| ---------------- | ------------------------- | ---------------------------------------------------------------- |
| 채널-공간 어텐션 | Channel Prior CPCA (2023) | CPCA 모듈: 채널 → 공간 어텐션 순차 적용 + [3,5,7,11] 커널 브랜치 |

**방법론 전제:** 고정된 크롭 크기 (ACDC: 224×224, ISIC: 512×512), 계산 효율성 유지

#### 2.9 계층적 3D FCN 계열

**계열 정의:** Coarse-to-fine 2 단계 파이프라인으로 정밀도 및 전체 시야 모두 확보

**공통 특징:**

- 3D U-Net 인코더-디코더
- 스킵 연결
- 가중치 손실 함수 (클래스 불균형 대응)
- 슬라이딩 윈도우 타일링

**주요 논문:**

| 방법론 계열   | 논문명                  | 핵심 특징                                        |
| ------------- | ----------------------- | ------------------------------------------------ |
| 계층적 3D FCN | Cascaded 3D FCNs (2018) | C₁ 전체 시야 → C₂ 정밀 영역 2 단계 파이프라인    |
| 3D U-Net 기반 | Deep Learning (2018)    | 4 레벨 인코더-디코더 + Dice 손실 + 타일링 접근법 |

**방법론 전제:** GPU 메모리 제약으로 $64^3$ 서브볼륨 단위, 클래스 불균형 대비를 위한 가중치 손실 함수

### 3. 핵심 설계 패턴 분석

#### 3.1 인코더-디코더 구조의 변형

| 패턴                 | 설명                             | 적용 논문                                     |
| -------------------- | -------------------------------- | --------------------------------------------- |
| 표준 U-Net           | 4 레벨 인코더/디코더 + 스킵 연결 | Deep Learning (2018), Cascaded 3D FCNs (2018) |
| 하이브리드 어텐션    | CNN 인코더 + Transformer 디코더  | MedNeXt (2024)                                |
| 비대칭 인코더-디코더 | MAE 학습용 전용 구조             | Self Pre-training (2022)                      |
| 듀얼 브랜치          | 공유 인코더 + 2 독립 디코더      | Scribble-Supervised (2022)                    |

#### 3.2 손실 함수 설계 패턴

| 손실 유형     | 공식                                                 | 적용 논문                            |
| ------------- | ---------------------------------------------------- | ------------------------------------ |
| Dice loss     | $L_l = - \frac{2 \sum p_i r_i}{\sum p_i + \sum r_i}$ | Deep Learning (2018), MedNeXt (2024) |
| Cross-entropy | $L_{CE}$                                             | TTA (2020), BayeSeg (2023)           |
| 혼합 손실     | $L_{total} = 0.5(L_{pCE} + L_{PLS})$                 | Scribble-Supervised (2022)           |
| 가중치 CE     | $L_k = (1 - N_k/N)/(C_K-1)$                          | Cascaded 3D FCNs (2018)              |

#### 3.3 적응 전략 패턴

| 적응 유형        | 기작                                | 적용 논문                  |
| ---------------- | ----------------------------------- | -------------------------- |
| 테스트-시간 적응 | I2NI 파라미터 적응 + DAE 타당성     | TTA (2020)                 |
| 파인튜닝         | 전체/파라미터 효율적 파인튜닝       | SAM 서베이 (2024)          |
| PEFT 어댑터      | LoRA/Adapter 최소 파라미터 업데이트 | Medical SAM Adapter (2023) |
| 자동 프롬프팅    | 자동 생성 프롬프트                  | SAM 서베이 (2024)          |

#### 3.4 예측 통합 패턴

| 통합 방식 | 설명                             | 적용 논문               |
| --------- | -------------------------------- | ----------------------- |
| 가산 통합 | $y = \sum \alpha_i y_i$ (α 합 1) | G-CASCADE (2023)        |
| 투표 전략 | 가우시안 가중치 투표             | Cascaded 3D FCNs (2018) |
| 다상 선택 | DGS 최적화 마스크 선택           | SAM Liver Tumor (2023)  |

### 4. 방법론 비교 분석

#### 4.1 문제 접근 방식 차이

| **도메인 적응**      | 테스트-시간 시프트 대응 | 테스트 이미지당 라벨 없음   |

#### 4.4 트레이드오프 분석

| 축              | CNN 중심            | Transformer 중심    | 하이브리드       |
| --------------- | ------------------- | ------------------- | ---------------- |
| **성능-용량**   | 높은 용량/낮은 효율 | 낮은 용량/높은 효율 | 균형             |

| **학습 비용**   | 낮음                | 높음                | 중간             |
| **일반화**      | 도메인 의존적       | 전이 학습 가능      | 도메인 적응 가능 |

### 5. 방법론 흐름 및 진화

#### 5.1 초기 CNN 기반 (2018)

- 3D U-Net 기반 계층적 구조 도입
- Dice 손실과 데이터 증강으로 90% Dice 달성
- 슬라이딩 윈도우 타일링로 GPU 메모리 제약 극복

#### 5.2 적응 전략 등장 (2020)

- 테스트-시간 적응으로 도메인 시프트 대응
- 자동 데이터 증강으로 하이퍼파라미터 자동 탐색
- 약지도 학습으로 주석 비용 절감

#### 5.3 Transformer 등장 및 발전 (2022-2023)

- ViT 기반 자기 사전 훈련으로 도메인 격차 해소
- SAM 파운데이션 모델 제로샷 일반화 가능
- PEFT/LoRA로 최소 파라미터 적응

#### 5.4 하이브리드 및 효율화 (2023-2024)

- 채널-공간 어텐션으로 CNN+Transformer 장점 결합
- 그래프 합성콜로 전역/지역 정보 동시 학습
- MedNeXt로 ConvNeXt 완전 활용 및 스케일링

### 6. 종합 정리

의료 영상 분할 방법론은 **(1) 아키텍처 설계**, **(2) 손실 함수 최적화**, **(3) 도메인 적응 전략** 3 축으로 구조화된다. CNN 기반 계층적 구조가 기본 골격을 제공한 뒤, Transformer가 전역 컨텍스트를 보완하며, 적응 전략이 도메인 격차를 해결하는 진화 경향이 나타난다.

**방법론의 축:**

- **모델 축:** CNN → Transformer → 하이브리드
- **데이터 축:** 강지도 → 약지도 → 제로샷
- **적응 축:** 고정 → 테스트-시간 → 파운데이션

제공된 자료 범위 내에서 파악된 방법론 지지는 **CNN 기반 구조**가 기본으로 **Transformer 기반 적응**을 결합하는 방향으로 진화하며, **계산 효율성 극대화**와 **일반화 능력 향상**이 지속적인 목표임을 보여준다.

## 3 장. 실험결과 분석

### 1. 평가 구조 및 공통 실험 설정

**주요 데이터셋 유형**

- **3D 볼륨 데이터**: Brain Tumor, Heart, Prostate, Pancreas, Abdominal organs, Lung tumors, Multi-phase liver tumors
- **2D 슬라이스**: ChestX-ray, Dermatology, Retinal vessels, Endoscopic images, Histopathology (WSI)
- **MRI 도메인**: Heart MRI (ACDC, RVSC, LA), Brain MRI (BraTS, HCP), Prostate MRI (NCI, PROMISE12), Abdominal CT (BTCV, AMOS)

**평가 환경 분류**

| 환경 유형           | 적용 논문                                         |
| ------------------- | ------------------------------------------------- |
| 시뮬레이션/벤치마크 | 모든 논문 (MSD Decathlon, BraTS 등 공개 데이터셋) |
| 교차 도메인 일반화  | Test-Time Adaptation, BayeSeg, MedNeXt            |
| 반지도/유기 학습    | Scribble-supervised, ACTION++, SAM 적응           |

**주요 평가 지표**

| 지표                                      | 적용 빈도 | 용도                                    |
| ----------------------------------------- | --------- | --------------------------------------- |
| Dice Similarity Coefficient (DSC)         | 매우 높음 | 기본 분할 성능                          |
| 95th Percentile Hausdorff Distance (HD95) | 높음      | 경계 정밀도                             |
| Accuracy                                  | 낮음      | 클래스 불균형 환경에서는 신뢰할 수 없음 |
| AUC                                       | 중간      | 분류 태스크 (ChestX-ray 등)             |
| Surface Dice (SDC)                        | 낮음      | 1mm 허용 오차 기준                      |
| IoU                                       | 중간      | 대체 분할 지표                          |
| ASD (Average Surface Distance)            | 중간      | 평균 표면 거리                          |

### 3. 성능 패턴 및 경향 분석

**도메인 일반화 및 데이터 효율성**

| 조건          | 성능 경향                               | 적용 논문                     |
| ------------- | --------------------------------------- | ----------------------------- |
| 레이블 비율 ↓ | 5-10% 레이블에서도 87-90% DSC 달성 가능 | ACTION++, Scribble-supervised |
| 도메인 시프트 | 아틀라스 기반 초기화, TTA 필요          | TTA, BayeSeg                  |
| 클래스 불균형 | Accuracy 는 신뢰할 수 없음, DSC 필수    | TTA, MIS Guidelines, BayeSeg  |
| 작은 ROI      | Accuracy 과대평가, 시각화 필수          | MIS Guidelines                |
