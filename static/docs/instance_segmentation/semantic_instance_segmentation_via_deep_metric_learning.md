# Semantic Instance Segmentation via Deep Metric Learning

- **저자**: Alireza Fathi, Zbigniew Wojna, Vivek Rathod, Peng Wang, Hyun Oh Song, Sergio Guadarrama, Kevin P. Murphy
- **발표연도**: 2017
- **arXiv**: https://ar5iv.labs.arxiv.org/html/1703.10277v1

## 1. 논문 개요

이 논문은 이미지 내에서 객체 인스턴스 분할(semantic instance segmentation)을 위한 새로운 방법을 제안한다. 연구의 목표는 단순히 픽셀을 분류하거나 바운딩 박스를 예측하는 것을 넘어, 이미지 내의 각 객체 인스턴스를 개별적으로 식별하고 그 형태를 마스크로 나타내는 것이다. 이는 자율주행, 로봇 공학, 사진 편집 등 다양한 실제 응용 분야에서 중요한 문제이다. 기존의 객체 감지 기반 방식이나 '박스 없는' 방식의 한계를 극복하고, 픽셀 간의 유사도를 기반으로 인스턴스를 분할하는 접근 방식을 제시한다.

## 2. 핵심 아이디어

이 논문의 핵심 아이디어는 두 픽셀이 동일한 객체 인스턴스에 속할 가능성을 계산하고, 이 유사도를 기반으로 유사한 픽셀들을 함께 그룹화하여 인스턴스 분할을 수행하는 것이다. 이는 비지도 이미지 분할과 유사하지만, 이 연구에서는 '정확한' 세그먼트에 대한 명확한 정의(객체 전체의 공간적 범위)를 가지고 있어 비지도 방식의 모호성을 피한다.

기존 접근 방식과의 차별점은 다음과 같다:
- **유사도 학습**: 딥 임베딩 모델을 사용하여 픽셀의 유사도 메트릭을 학습한다. 이는 FaceNet과 같이 바운딩 박스의 유사도를 학습하는 방식과 달리, 픽셀 단위의 유사도를 로컬 컨텍스트를 고려하여 학습한다.
- **그룹화 방법**: 모든 픽셀 쌍의 유사도를 계산하는 것은 비용이 너무 크기 때문에, 스펙트럼(spectral) 또는 그래프 기반(graph-based) 분할 방법을 사용하지 않는다. 대신, 미리 선택된 $K$개의 "시드 포인트(seed points)"에 대한 임베딩 공간에서의 거리를 계산하여 그룹화를 수행한다.
- **시드 포인트 선정**: 픽셀이 좋은 시드 포인트가 될 가능성을 예측하는 별도의 모델을 학습하여 "시드니스(seediness)" 점수를 부여한다. 이 시드니스 점수는 기존 방법의 "중심성(centeredness)" 점수와 유사하지만, 객체의 중심을 식별하는 대신 인스턴스 내 다른 픽셀에 대한 픽셀의 "전형성(typicality)"을 측정한다.
- **임베딩 차원**: 기존 연구([19] 등)에서 1차원 임베딩을 사용한 것과 달리, 이 논문에서는 $D$차원 임베딩(실험에서는 64차원)을 사용하여 더 나은 결과를 얻는다.

## 3. 상세 방법 설명

제안하는 방법은 의미론적 분할을 위해 사전 학습된 모델을 기반으로 하며, 인스턴스 분할을 위해 두 가지 종류의 출력 "헤드"를 추가하여 수정한다.

**3.1 접근 방식 개요**
- **임베딩 헤드**: 각 픽셀에 대한 임베딩 벡터를 생성한다. 이상적으로는 유사한 임베딩 벡터를 가진 픽셀들이 동일한 객체 인스턴스에 속할 가능성이 높다.
- **분류 및 시드니스 헤드**: 각 픽셀을 중심으로 하는 마스크에 대한 클래스 레이블과 해당 픽셀이 마스크를 생성하기 위한 좋은 "시드"가 될 수 있는지에 대한 신뢰도 점수("시드니스" 점수)를 예측한다.

**3.2 임베딩 모델**
네트워크의 임베딩 헤드는 컨볼루션 특징 추출기(3.5절 참조)에서 얻은 특징 맵을 입력으로 받는다. 출력은 $[h, w, d]$ 텐서이며, 여기서 $h$는 이미지 높이, $w$는 이미지 너비, $d$는 임베딩 공간 차원이다(실험에서는 64차원 임베딩 사용). 따라서 이미지의 각 픽셀 $p$는 $d$차원 임베딩 벡터 $e_p$로 표현된다.

임베딩 벡터가 주어지면, 픽셀 $p$와 $q$ 사이의 유사도 $\sigma(p,q)$는 다음과 같이 계산된다:

$$\sigma(p,q)=\frac{2}{1+\exp(||e_{p}-e_{q}||_{2}^{2})}\quad(1)$$

이 식에 따르면 임베딩 공간에서 가까운 픽셀 쌍은 $\sigma(p,q)=1$의 유사도를 가지고, 멀리 떨어진 픽셀 쌍은 $\sigma(p,q)=0$의 유사도를 갖는다.

네트워크는 다음 손실 함수를 최소화하여 훈련된다:

$$\mathcal{L}\_{e} = -\frac{1}{|S|}\sum_{p,q\in S}w_{pq}\left[1_{\{y_{p}=y_{q}\}}\log(\sigma(p,q)) + 1_{\{y_{p}\
eq y_{q}\}}\log(1-\sigma(p,q))\right]$$

여기서 $S$는 선택된 픽셀 집합, $y_p$는 픽셀 $p$의 인스턴스 레이블, $w_{pq}$는 가중치이다.

**3.3 그룹화(Clustering) 모델**
시드 포인트는 시드니스 점수와 겹치지 않는 마스크를 생성하는 비최대 억제(NMS)를 통해 선택된다. 그런 다음 각 시드 포인트는 임베딩 공간에서 유클리드 거리를 기반으로 픽셀을 그룹화하는 데 사용된다.

**3.4 분류 및 시드니스 모델**
분류 헤드는 각 시드 포인트에 대한 객체 클래스 레이블과 해당 시드 포인트가 좋은 마스크를 생성할 가능성을 나타내는 시드니스 점수를 예측한다. 시드니스 점수는 네트워크의 다른 헤드에서 출력된 특징 맵에서 파생된다.

**3.5 공유된 전체 이미지 컨볼루션 특징**
이 모델은 의미론적 분할을 위해 사전 학습된 ResNet-101을 기반으로 하며, 두 개의 새로운 출력 헤드를 추가한다.

## 4. 실험 및 결과

- **데이터셋 및 평가 지표**: Pascal VOC 2012 인스턴스 분할 벤치마크에서 평가가 이루어졌다. 주요 평가 지표는 IoU(Intersection over Union) 0.5에서 측정한 mAP(mean Average Precision)이다.
- **주요 결과**: 제안하는 방법은 Pascal VOC 2012에서 62.21%의 mAP 점수를 달성했다.
- **경쟁 분석**: 이 결과는 해당 벤치마크에서 네 번째 순위에 해당하며, [16] (66.7%), [15] (65.7%), [6] (63.5%)의 제안 기반(proposal-based) 방법들보다는 낮지만, 58.7%를 기록한 이전의 '제안 없는(proposal-free)' 방법([17])보다는 높은 수치이다. 비록 최첨단(state-of-the-art) 성능은 아니지만, 경쟁력 있는 결과를 보여준다.

## 5. 강점, 한계

- **강점**:
 - **연결되지 않은 영역 그룹화**: 기존의 watershed 알고리즘 기반 기술과 달리, 이 방법은 가려진 객체에 의해 두 부분으로 분할된 말과 같이 연결되지 않은 영역도 단일 인스턴스로 그룹화할 수 있다.
 - **유연성**: "가늘고 긴" 형태나 특이한 모양을 가진 객체가 있는 다른 데이터셋에서 특히 잘 작동할 가능성이 있다. (이 부분은 향후 연구로 남겨졌다.)
- **한계**:
 - **Pascal VOC 벤치마크에서 SOTA 아님**: 특정 벤치마크(Pascal VOC 2012)에서는 최첨단 성능을 달성하지 못했다.

## 6. 결론

이 논문은 픽셀 간의 유사도를 학습하고 이를 기반으로 그룹화하는 새로운 인스턴스 분할 방법을 제안한다. 딥 임베딩 모델과 시드니스 점수를 활용하여 효과적인 인스턴스 분할을 수행하며, Pascal VOC 2012 벤치마크에서 경쟁력 있는 결과를 입증했다. 이 연구는 연결되지 않은 객체 인스턴스를 처리하는 능력과 다양한 형태의 객체에 대한 잠재적 확장성을 보여주며, 향후 연구를 위한 흥미로운 방향을 제시한다.
