# Predicting Future Instance Segmentation by Forecasting Convolutional Features

- **저자**: Pauline Luc, Camille Couprie, Yann LeCun, Jakob Verbeek
- **발표연도**: 2018
- **arXiv**: https://ar5iv.labs.arxiv.org/html/1803.11496v2

## 1. 논문 개요

이 논문의 목표는 미래의 인스턴스 세그멘테이션(instance segmentation)을 예측하는 새로운 문제를 다루는 것입니다. 지능적인 행동을 위해서는 미래 사건을 예측하는 능력이 중요하며, 비디오 예측은 이러한 목표를 위한 대리 작업으로 연구되어 왔습니다. 기존 연구에서는 미래 프레임의 시맨틱 세그멘테이션(semantic segmentation)을 예측할 때, RGB 프레임을 예측한 다음 이를 분할하는 것보다 시맨틱 레벨에서 직접 예측하는 것이 더 효과적이라는 것을 보여주었습니다. 이 논문은 개별 객체를 추가적으로 분할하는 더 도전적인 문제인 미래 인스턴스 세그멘테이션을 목표로 합니다.

## 2. 핵심 아이디어

논문의 핵심 아이디어는 미래 인스턴스 세그멘테이션을 예측하기 위해 Mask R-CNN 모델의 고정 크기 컨볼루션 특징(convolutional features) 공간에서 예측 모델을 개발하는 것입니다. 기존의 시맨틱 세그멘테이션 예측 방법은 프레임 간 인스턴스 레이블 수가 변하고 비디오 시퀀스 전반에 걸쳐 일관성이 없기 때문에 인스턴스 세그멘테이션에 직접 적용하기 어렵습니다. 이를 해결하기 위해, 저자들은 미래 프레임의 RGB 픽셀 값이나 직접적인 인스턴스 레이블 대신, Mask R-CNN의 컨볼루션 특징을 예측합니다. 예측된 특징에 Mask R-CNN의 "탐지 헤드(detection head)"를 적용하여 미래 프레임의 인스턴스 세그멘테이션을 생성합니다. 이는 가변적인 출력 레이블 수를 처리하면서도 의미론적으로 풍부한 예측을 가능하게 합니다.

## 3. 상세 방법 설명

이 연구의 방법론은 Mask R-CNN의 구조를 기반으로 하며, 미래 프레임의 컨볼루션 특징을 예측하는 "Feature to Feature (F2F)" 네트워크를 통합합니다.

- **Mask R-CNN 개요**: Mask R-CNN은 세 가지 주요 단계로 구성됩니다.
  1. **CNN 백본(Backbone)**: 입력 이미지에서 고수준 특징 맵을 추출합니다. 이 논문에서는 Feature Pyramid Network (FPN)을 백본으로 사용합니다. FPN은 여러 공간 해상도에서 특징을 추출하여 다양한 스케일의 객체를 탐지할 수 있도록 합니다.
  2. **영역 제안 네트워크 (Region Proposal Network, RPN)**: 백본에서 추출된 특징을 사용하여 인스턴스를 포함할 가능성이 있는 바운딩 박스(bounding box) 형태의 관심 영역(Regions of Interest, ROIs)을 제안합니다.
  3. **탐지 헤드(Detection Head)**: 각 ROI의 특징을 입력받아 정제된 바운딩 박스 좌표, 클래스 예측, 그리고 예측된 클래스에 대한 고정 크기 이진 마스크를 생성합니다. 마지막으로 마스크는 예측된 바운딩 박스 내에서 전체 이미지 해상도로 다시 보간되어 최종 인스턴스 세그멘테이션으로 보고됩니다.

- **컨볼루션 특징 예측(Forecasting Convolutional Features)**:
  - **문제 정의**: 주어진 비디오 시퀀스에서 하나 이상의 미래 프레임에 대한 인스턴스 레벨 객체 세그멘테이션을 예측하는 것이 목표입니다. 여기서 미래 프레임의 RGB 픽셀 값은 접근할 수 없습니다.
  - **고정 크기 특징의 활용**: 인스턴스 세그멘테이션 자체는 프레임 간 인스턴스 레이블 수의 변화와 비디오 시퀀스 간 불일치로 인해 예측에 적합한 표현을 제공하지 않습니다. 이를 해결하기 위해, Mask R-CNN 아키텍처에서 고정된 크기를 가지는 최고 수준의 특징을 예측하는 방식을 채택합니다. 특히 FPN 백본을 사용하여 과거 프레임 $X_{t-\tau}$부터 $X_t$까지의 특징 피라미드가 주어졌을 때, 관측되지 않은 미래 프레임 $X_{t+1}$의 특징 피라미드를 예측하는 모델을 학습합니다.
  - **멀티 스케일 접근 방식**: 다른 FPN 레벨의 특징들은 공간-시간 역학이 다르기 때문에, 각 레벨에 대해 별도의 네트워크를 사용하여 특징을 예측하는 멀티 스케일 접근 방식을 제안합니다. 각 레벨의 예측 네트워크는 완전히 독립적으로 훈련되며, 이를 "Feature to Feature (F2F)" 모델 (F2F$_l$)이라고 부릅니다. 이 방식은 훈련을 여러 GPU에 걸쳐 병렬화할 수 있게 합니다.
  - **F2F 네트워크 구조**: 각 F2F$_l$ 네트워크는 해상도를 보존하는 CNN으로 구현됩니다. 이는 시야(field of view)를 효율적으로 넓히면서 고해상도 디테일을 유지하기 위해 멀티 스케일 구조를 가집니다.

## 4. 실험 및 결과

이 논문은 광학 흐름(optical flow) 기반 및 재구성된 인스턴스 세그멘테이션 아키텍처 기반의 강력한 기준선들과 비교하여 제안하는 접근 방식의 성능을 평가했습니다. 실험 결과, 컨볼루션 특징을 예측하는 이 접근 방식이 미래 인스턴스 세그멘테이션 예측에서 기존의 강력한 기준선들에 비해 성능을 크게 향상시킴을 보여주었습니다. 특히, 인스턴스 모델링이 개별 보행자의 세그멘테이션 정확도를 현저히 개선한다는 것을 입증했습니다.

## 5. 강점, 한계

**강점**:

- **가변 크기 출력 처리**: 객체 탐지 및 인스턴스 세그멘테이션과 같이 모델 출력이 가변적인 경우를 처리할 수 있습니다.
- **비디오 시퀀스 레이블 불필요**: 훈련을 위해 레이블이 지정된 비디오 시퀀스가 필요하지 않습니다. 중간 CNN 특징 맵은 레이블이 지정되지 않은 데이터에서 직접 계산할 수 있습니다.
- **다양한 장면 해석 지원**: 표면 법선(surface normals), 객체 바운딩 박스, 인간 신체 부위 레이블과 같은 다양한 장면 해석을 생성할 수 있는 모델을 지원합니다. 이 모든 작업에 대한 적절한 인코더와 손실 함수를 설계할 필요가 없습니다.
- **의미론적 풍부함**: 기존의 예측 인식 작업보다 의미론적으로 더 풍부한 미래 인스턴스 세그멘테이션이라는 새로운 작업을 도입합니다.

**한계**:
논문에서 명시적으로 한계를 언급하지는 않았지만, 각 FPN 레벨에 대해 별도의 F2F 네트워크를 훈련하는 멀티 스케일 접근 방식은 모델의 복잡성을 증가시키고, 다른 해상도 간 예측을 연결하는 대안적인 아키텍처에 대한 탐구는 논문의 범위를 벗어난다고 언급되어, 이는 향후 연구의 방향이 될 수 있음을 시사합니다.

## 6. 결론

이 논문은 미래 인스턴스 세그멘테이션 예측이라는 새로운 과제를 제시하고, 이를 해결하기 위한 혁신적인 자기 지도 학습(self-supervised learning) 접근 방식을 제안했습니다. 핵심 기여는 Mask R-CNN의 고수준 컨볼루션 특징을 예측하여 미래 인스턴스 세그멘테이션을 수행하는 것입니다. 이 방법은 가변적인 인스턴스 레이블 문제를 해결하고, 레이블 없는 비디오 데이터를 활용하며, 다양한 인식 작업을 지원할 수 있습니다. 실험을 통해 제안된 특징 학습 접근 방식이 광학 흐름 및 재구성된 인스턴스 세그멘테이션 아키텍처를 기반으로 한 강력한 기준선보다 성능이 우수함을 입증했습니다. 이 연구는 자율 주행과 같은 의사 결정이 중요한 분야에서 미래 상황에 대한 정밀한 이해를 가능하게 하여 실제 적용 및 향후 연구에 중요한 영향을 미 미칠 잠재력을 가지고 있습니다.
