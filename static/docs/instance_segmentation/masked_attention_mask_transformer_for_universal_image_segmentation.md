# Masked-attention Mask Transformer for Universal Image Segmentation

- **저자**: Bowen Cheng, Ishan Misra, Alexander G. Schwing, Alexander Kirillov, Rohit Girdhar
- **발표연도**: 2021
- **arXiv**: https://ar5iv.labs.arxiv.org/html/2112.01527v3

## 1. 논문 개요

이 논문은 이미지 분할(Image Segmentation) 문제에 초점을 맞추고 있으며, 특히 범용 이미지 분할 아키텍처인 Masked-attention Mask Transformer (Mask2Former)를 제안한다. 이미지 분할은 픽셀들을 다양한 의미(예: 카테고리 또는 인스턴스 멤버십)에 따라 그룹화하는 것을 목표로 한다. 기존 연구는 각 분할 작업(파노라마, 인스턴스, 의미론적 분할)마다 특화된 아키텍처를 설계하는 데 중점을 두었으나, 이는 연구 노력의 중복을 초래하고 유연성이 부족하다는 한계가 있었다. Mask2Former는 이러한 문제점을 해결하기 위해 모든 이미지 분할 작업을 처리할 수 있는 단일 아키텍처를 제시하며, 기존의 특화된 아키텍처들을 능가하는 성능을 달성하는 것을 목표로 한다.

## 2. 핵심 아이디어

Mask2Former의 핵심 아이디어는 `masked attention` 메커니즘을 Transformer 디코더에 도입하여 이미지 분할 작업의 효율성과 성능을 개선하는 것이다. 기존 Transformer 디코더의 크로스 어텐션(cross-attention)이 이미지 내의 모든 위치에 주의를 기울이는 것과 달리, Mask2Former의 `masked attention`은 예측된 마스크 영역 내의 전경 영역에만 어텐션을 제한하여 지역화된 특징을 추출한다. 이는 더 빠른 수렴과 향상된 성능으로 이어진다. 또한, 작은 객체를 분할하기 위해 멀티 스케일 고해상도 특징을 활용하고, 학습 가능한 쿼리 특징, 드롭아웃 제거 등 최적화 개선을 통해 추가적인 계산 없이 성능을 향상시킨다. 이로써 단일 아키텍처로 다양한 분할 작업을 해결하면서도 학습 효율성을 높여 범용 아키텍처의 접근성을 향상시킨다.

## 3. 상세 방법 설명

Mask2Former는 기존 MaskFormer [14]와 유사한 메타 아키텍처를 기반으로 한다. 이 아키텍처는 다음 세 가지 주요 구성 요소로 이루어진다.

- **백본(Backbone)**: 이미지로부터 저해상도 특징을 추출한다.
- **픽셀 디코더(Pixel Decoder)**: 백본의 출력에서 저해상도 특징을 점진적으로 업샘플링하여 고해상도 픽셀별 임베딩(per-pixel embeddings)을 생성한다.
- **트랜스포머 디코더(Transformer Decoder)**: 이미지 특징에 작용하여 객체 쿼리(object queries)를 처리한다. 최종 이진 마스크 예측은 객체 쿼리와 픽셀별 임베딩으로부터 디코딩된다.

Mask2Former는 특히 트랜스포머 디코더에 다음과 같은 주요 개선 사항을 적용한다.

### 3.2.1 Masked attention

기존의 크로스 어텐션은 전체 특징 맵에 대해 어텐션을 수행하여 학습 수렴이 느려지는 문제가 있었다. Mask2Former는 각 쿼리에 대해 예측된 마스크의 전경 영역 내에서만 어텐션을 제한하는 `masked attention`을 제안한다. 이는 지역화된 특징을 추출하고, 컨텍스트 정보는 self-attention을 통해 얻을 수 있다는 가설에 기반한다.

### 3.2.2 고해상도 특징 활용

작은 객체들을 효과적으로 처리하기 위해, Mask2Former는 픽셀 디코더의 특징 피라미드에서 생성된 다양한 스케일의 특징 맵을 트랜스포머 디코더의 각 레이어에 순환적으로(round robin fashion) 공급하는 효율적인 멀티 스케일 전략을 사용한다.

### 3.2.3 최적화 개선

성능을 향상시키면서도 추가적인 계산량을 발생시키지 않기 위해 다음과 같은 최적화 개선이 이루어졌다.

- Self-attention과 cross-attention의 순서를 변경한다.
- 쿼리 특징(query features)을 학습 가능하게 만든다.
- 드롭아웃(dropout)을 제거한다.
- 마스크 손실(mask loss)을 무작위로 샘플링된 적은 수의 점들(few randomly sampled points)에서 계산하여 학습 메모리를 약 3배 절약한다.

이러한 개선 사항들은 모델의 성능을 향상시킬 뿐만 아니라 학습을 훨씬 쉽게 만들어, 제한된 컴퓨팅 자원을 가진 사용자들도 범용 아키텍처에 더 쉽게 접근할 수 있도록 한다.

## 4. 실험 및 결과

Mask2Former는 세 가지 이미지 분할 작업(파노라마, 인스턴스, 의미론적 분할)에 대해 네 가지 인기 데이터셋(COCO [35], Cityscapes [16], ADE20K [65], Mapillary Vistas [42])을 사용하여 평가되었다.

실험 결과, Mask2Former는 처음으로 모든 벤치마크에서 특화된 아키텍처와 동등하거나 더 나은 성능을 달성했다.

- **COCO 파노라마 분할**: 57.8 PQ로 새로운 최신(state-of-the-art) 성능을 달성했다.
- **COCO 인스턴스 분할**: 50.1 AP로 새로운 최신 성능을 달성했다.
- **ADE20K 의미론적 분할**: 57.7 mIoU로 새로운 최신 성능을 달성했다.

이러한 결과는 Mask2Former가 단일 아키텍처로 다양한 이미지 분할 작업에서 우수한 일반화 성능을 보여주며, 각 작업에 특화된 최고 성능의 아키텍처들을 능가한다는 것을 입증한다.

## 5. 강점, 한계

**강점**:

- **범용성**: 파노라마, 인스턴스, 의미론적 분할을 포함한 모든 이미지 분할 작업을 단일 아키텍처로 처리할 수 있다.
- **최첨단 성능**: 여러 인기 데이터셋에서 기존의 특화된 아키텍처보다 뛰어나거나 동등한 최신 성능을 달성한다.
- **효율적인 학습**: `masked attention`, 멀티 스케일 고해상도 특징 활용, 최적화 개선을 통해 학습 수렴 속도가 빠르고 메모리 효율적이며, 제한된 컴퓨팅 자원을 가진 사용자도 접근하기 용이하다.

**한계**:
논문에서 Mask2Former 자체의 명시적인 한계를 직접적으로 언급하지는 않는다. 하지만, 기존 범용 아키텍처들이 겪었던 문제점(특정 작업에서의 성능 저하, 학습의 어려움)을 Mask2Former가 해결했음을 강조한다. 따라서 Mask2Former의 한계는 이러한 문제점들이 완전히 해소되었는지에 대한 추가적인 연구가 필요할 수 있다.

## 6. 결론

이 연구는 `Masked-attention Mask Transformer (Mask2Former)`라는 새로운 범용 이미지 분할 아키텍처를 성공적으로 제안했다. Mask2Former는 `masked attention` 메커니즘과 멀티 스케일 고해상도 특징 활용, 그리고 다양한 최적화 개선을 통해 기존의 특화된 아키텍처들을 능가하는 성능을 달성했다. 또한, 단일 아키텍처로 파노라마, 인스턴스, 의미론적 분할 등 모든 이미지 분할 작업에서 최첨단 결과를 보여주며, 학습 효율성까지 개선하여 범용 아키텍처의 실용적인 적용 가능성을 크게 높였다. 이 연구는 이미지 분할 분야에서 범용적이고 효율적인 모델 개발의 중요한 진전을 의미하며, 향후 다양한 컴퓨터 비전 응용 분야에 중요한 영향을 미칠 것으로 기대된다.
