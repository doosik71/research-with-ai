# Recurrent Neural Networks for Semantic Instance Segmentation

- **저자**: Amaia Salvador, Míriam Bellver, Víctor Campos, Manel Baradad, Ferran Marques, Jordi Torres, Xavier Giro-i-Nieto
- **발표연도**: 2017
- **arXiv**: https://ar5iv.labs.arxiv.org/html/1712.00617v4

## 1. 논문 개요

이 논문은 이미지 내의 모든 객체에 대해 이진 마스크와 해당 클래스 확률을 순차적으로 생성하는 의미론적 인스턴스 분할(Semantic Instance Segmentation)을 위한 순환 모델을 제안한다. 기존의 객체 제안(object proposal) 기반 방법론이 복잡한 후처리 단계를 요구하는 것과 달리, 제안된 시스템은 입력 이미지로부터 레이블이 지정된 마스크 시퀀스를 end-to-end 방식으로 학습할 수 있다. 이 연구는 인간의 시각적 탐색 과정을 모방하여 장면의 순차적 분석을 수행하며, 이를 통해 복잡한 객체 분포를 처리하고 상호 일관된 예측을 수행한다. 논문의 주요 기여는 다음과 같다: (a) 의미론적 인스턴스 분할을 위한 최초의 end-to-end 순환 모델 제시, (b) 세 가지 인스턴스 분할 벤치마크에서 기존 순차적 방법론 대비 경쟁력 있는 성능 입증, (c) 모델이 따르는 객체 발견 패턴에 대한 심층 분석.

## 2. 핵심 아이디어

핵심 아이디어는 의미론적 인스턴스 분할 작업을 순환 신경망(RNN)을 활용한 시퀀스 예측 작업으로 공식화하고 해결하는 것이다. 기존의 객체 제안 기반 방법들이 과도한 수의 예측을 생성하고 비최대 억제(non-maximum suppression)와 같은 외부 후처리 단계를 필요로 하는 반면, 이 모델은 이미지 픽셀로부터 최종 의미론적 인스턴스 분할을 직접 예측하여 이러한 문제를 회피한다.

이 모델은 기존의 순차적 방법론(예: Ren & Zemel, Romera-Paredes & Torr)과 달리, 완전한 end-to-end 방식으로 이미지 내 모든 객체에 대한 이진 마스크와 범주형 레이블을 모두 출력한다. 이전 순차적 방법들은 종종 클래스에 구애받지 않거나 클래스 확률을 위해 별도의 모델을 필요로 했다.

제안된 순환 아키텍처는 입력 이미지에 따라 달라지는 가변 길이 출력을 처리할 수 있도록, 마스크와 레이블 시퀀스를 한 번에 하나의 인스턴스씩 출력한다. 각 예측 $\hat{y}_t$는 이진 마스크 $\hat{y}_m \in [0,1]^{h \times w}$, 이미지 차원으로 정규화된 바운딩 박스 좌표 $\hat{y}_b \in [0,1]^4$, $C$개의 다른 범주에 대한 확률 $\hat{y}_c \in [0,1]^C$, 그리고 테스트 시 중단 기준이 되는 객체성 점수 $\hat{y}_s \in [0,1]$를 포함한다. 바운딩 박스 주석을 분할 마스크로부터 얻는 것은 간단하며, 이는 실험에서 더 나은 성능을 내는 추가 학습 신호로 작용한다.

## 3. 상세 방법 설명

이 모델은 의미론적 분할 작업에서 흔히 사용되는 인코더-디코더 아키텍처를 따른다. 인코더의 레이어로부터 스킵 연결(skip connections)을 사용하여 정확한 분할 출력을 얻는 데 유용한 저수준 특징(low-level features)을 복구한다. 이 모델과 기존 작업의 주요 차이점은 디코더가 순환적이라는 점이다. 이는 모든 객체가 존재하는 단일 의미론적 분할 맵 대신 한 번에 하나의 인스턴스를 예측할 수 있게 하여, 가변 길이 출력을 자연스럽게 처리할 수 있도록 한다.

(참고: 웹페이지 내용이 잘려서 인코더 및 모델의 다른 세부 구성 요소에 대한 상세 정보는 이 요약에 포함되지 않았습니다.)

## 4. 실험 및 결과

이 논문은 제안된 순환 모델의 적합성을 Pascal VOC 2012, CVPPP Plant Leaf Segmentation, 그리고 Cityscapes의 세 가지 인스턴스 분할 벤치마크에서 연구했다. 모델이 생성하는 객체 정렬 패턴을 분석한 결과, 인코더 네트워크에서 학습된 활성화와 상관관계가 있는 일관된 패턴을 따르는 것을 관찰했다. 이는 모델이 이미지를 탐색하는 방식을 학습한다는 것을 의미한다.

(참고: 웹페이지 내용이 잘려서 주요 정량적/정성적 결과의 구체적인 수치는 이 요약에 포함되지 않았습니다.)

## 5. 강점, 한계

**강점**:

- 의미론적 인스턴스 분할을 위한 최초의 end-to-end 순환 모델을 제안하여, 기존 객체 제안 기반 방법론의 복잡한 후처리 단계(예: NMS)를 제거했다.
- 이미지 픽셀로부터 직접 의미론적 인스턴스 분할을 예측하여 효율성을 높였다.
- 가변 길이 출력(인스턴스 수)을 자연스럽게 처리할 수 있는 구조를 갖췄다.
- 일관된 객체 발견 패턴을 학습하는 능력을 보여주었다.
- 바운딩 박스 주석을 추가 학습 신호로 활용하여 모델 성능을 향상시켰다.

**한계**:

- 웹페이지 내용이 잘려 모델 아키텍처(특히 인코더의 상세 기능 외의 부분)에 대한 완전한 정보가 제공되지 않아, 모델의 전체적인 복잡성이나 구현 세부 사항을 파악하기 어렵다.
- 구체적인 정량적 실험 결과가 제시되지 않아, "경쟁력 있는 성능"의 정확한 정도를 평가하기 어렵다.
- 논문이 "Anonymous ECCV submission"으로 언급되어, 최종 출판 상태나 학술 대회 정보가 현재 내용으로는 불분명하다.
- 이 요약은 제공된 웹페이지의 잘린 내용에 기반하므로, 논문의 모든 측면을 완전히 다루지 못할 수 있다.

## 6. 결론

이 연구는 의미론적 인스턴스 분할을 위한 혁신적인 end-to-end 순환 모델을 제시한다. 이 모델은 인스턴스 분할을 시퀀스 예측 작업으로 재정의함으로써, 기존의 객체 제안 기반 방법론과 그에 따른 후처리 필요성을 제거하는 새로운 접근 방식을 제공한다. 순환 모델이 이 작업에 효과적임을 입증하고, 객체 발견 패턴에 대한 통찰력을 제공한다. 이러한 접근 방식은 인스턴스 분할 파이프라인을 단순화하여 실제 적용 및 향후 연구에서 효율성과 정확성을 향상시킬 잠재력을 가지고 있다.
