# Bridging Category-level and Instance-level Semantic Image Segmentation

- **저자**: Zifeng Wu, Chunhua Shen, Anton van den Hengel
- **발표연도**: 2016
- **arXiv**: https://ar5iv.labs.arxiv.org/html/1605.06885v1

## 1. 논문 개요

이 논문은 범주 수준(category-level)의 의미론적 이미지 분할을 기반으로 하는 인스턴스 수준(instance-level) 이미지 분할 접근 방식을 제안한다. 연구 문제는 기존의 인스턴스 분할 접근 방식들이 주로 객체 탐지(detect-then-segment)에 의존하여 바운딩 박스를 먼저 예측한 후 마스크를 생성하는 방식이라는 점이다. 이러한 방식은 최신 의미론적 분할 모델의 강력한 기능을 완전히 활용하지 못한다는 한계가 있다. 이 연구의 중요성은 의미론적 분할의 강점을 활용하여 인스턴스 분할에서 기존의 탐지 기반 접근 방식들과 견줄 수 있거나 더 나은 성능을 달성하는 새로운 파이프라인을 제시하는 데 있다.

## 2. 핵심 아이디어

이 논문의 중심 아이디어는 각 의미론적 범주 마스크의 픽셀에 대해 해당 인스턴스의 바운딩 박스를 딥 완전 컨볼루션 회귀 네트워크(deep fully convolutional regression network)를 사용하여 예측함으로써 인스턴스 수준 분할을 수행하는 것이다. 이는 기존의 "탐지 후 분할(detect-then-segment)" 방식과는 다른 파이프라인을 따른다. 주요 차별점은 탐지 기반 접근 방식이 먼저 객체의 바운딩 박스를 예측하는 반면, 제안된 방법은 의미론적 범주 분할의 결과 위에 인스턴스 수준 분할을 구축하여 이전에 탐색되지 않았던 경로의 잠재력을 보여준다는 점이다.

## 3. 상세 방법 설명

이 논문에서 제안하는 인스턴스 분할 파이프라인은 다음과 같은 단계를 거친다 (Figure 1 참조):

1. **범주별 점수 맵 계산**: 의미론적 범주 분할을 통해 이미지 내 각 범주에 대한 점수 맵을 계산한다.
2. **범주별 변환 맵 계산**: 바운딩 박스 회귀를 통해 각 픽셀에서 해당 인스턴스의 바운딩 박스 중심까지의 수직/수평 오프셋과 인스턴스의 높이 및 너비를 포함하는 변환 맵을 계산한다.
3. **변환 맵 적용**: 얻은 변환 맵을 해당 점수 맵에 적용한다. 이 과정에서 배경 픽셀은 무시하고 전경 픽셀만 변환한다.
4. **지역 최대값 검색**: 변환된 맵에서 비최대 억제(NMS)를 통해 지역 최대값을 검색하고, 이를 탐지된 인스턴스 가설로 유지한다.
5. **마스크 복원**: 억제된 모든 픽셀을 추적하여 각 인스턴스 가설에 대한 마스크를 복원한다.
6. **최종 결과 생성**: 영역 기반 NMS를 사용하여 최종 인스턴스 분할 결과를 생성한다.

훈련 시에는 의미론적 분할 네트워크는 고전적인 로지스틱 회귀 손실로, 지역화 네트워크는 평활화된 $L_1$ 손실(smoothed $L_1$ loss)로 각각 훈련한다. 다양한 크기의 인스턴스 기여도를 균형 있게 맞추기 위해 훈련 손실에서 픽셀의 높이와 너비에 따라 가중치를 재조정하며, 동일한 인스턴스 내의 픽셀은 동일한 손실 가중치를 갖는다.

**온라인 부트스트래핑(Online Bootstrapping)**:
훈련 중 네트워크가 "어려운(hard)" 픽셀에 집중하도록 하는 온라인 부트스트래핑 방법을 제안한다. 의미론적 범주 수준 분할의 경우, 손실 함수는 다음과 같다:

$$
\ell = - \left( \frac{1}{\sum_{i=1}^N \sum_{j=1}^K \mathbb{1}\{y_i=j \text{ and } p_{ij} < t\}} \right) \cdot \left( \sum_{i=1}^N \sum_{j=1}^K \mathbb{1}\{y_i=j \text{ and } p_{ij} < t\} \log p_{ij} \right)
$$

여기서 $t$는 $(0,1]$ 범위의 임계값이며, $\mathbb{1}\{\cdot\}$는 조건이 참일 때 1, 그렇지 않을 때 0이다. 이 함수는 손실이 임계값 $t$보다 낮은(너무 쉬운) 픽셀을 무시한다. 임계값 $t$는 모델 성능에 따라 동적으로 조정되어, 미니 배치당 적절한 수의 픽셀을 유지하고 노이즈가 많은 기울기를 방지한다. 지역화 네트워크의 부트스트래핑도 유사하게 "쉬운" 픽셀을 제외하지만, 회귀 손실 대신 예측된 바운딩 박스와 실제 바운딩 박스 간의 IoU(Intersection-over-Union) 점수를 임계값으로 사용한다. 이 방법은 편향된 훈련 데이터(예: 배경 픽셀이 객체 픽셀보다 많거나 특정 범주가 지배적인 경우)의 균형을 맞추는 데 도움이 된다.

**완전 컨볼루션 잔여 네트워크(Fully Convolutional Residual Network, FCRN)**:
FCRN은 ResNet [6]을 기반으로 초기화된다. 선형 분류 계층은 공간 위치당 하나의 예측을 하는 컨볼루션 계층으로 대체된다. 특징 맵 해상도를 높이기 위해 7x7 풀링 계층은 제거되고, 딜레이션 컨볼루션(dilation convolution, 또는 atrous algorithm)을 도입한다. 딜레이션 컨볼루션은 커널을 확장하여 매개변수 수를 크게 늘리지 않고도 더 큰 수용 필드(Field-of-View, FoV)를 달성하면서 고해상도 특징 맵을 유지할 수 있게 한다.

## 4. 실험 및 결과

이 논문은 PASCAL VOC 2012 데이터셋에서 의미론적 분할 및 인스턴스 분할 모두에 대한 실험을 수행했다.

- **의미론적 분할**: FCRN의 다양한 변형을 광범위하게 평가하여 계층 수, 특징 맵 해상도, 수용 필드 크기를 포함한 최상의 구성을 찾았다. 또한 FCRN 잔여 블록에 드롭아웃 정규화를 도입하고, 최상위 선형 분류기를 다중 계층 비선형 분류기로 교체하며, 다중 뷰 테스트 기법을 채택하여 성능을 더욱 향상시켰다. 그 결과, 증강된 PASCAL VOC 훈련 데이터만 사용하여 79.1%의 평균 IoU(intersection-over-union) 점수를 달성했으며, 이는 당시 PASCAL VOC 2012 데이터셋에서 최고 기록이었다.
- **인스턴스 분할**: PASCAL VOC 2012 데이터셋에서 이전 최고 성능과 비교하여 동등하거나 더 나은 결과를 달성했다. 특히, 겹침(overlap) 0.7에서의 평균 영역 평균 정밀도(mean region average precision)를 41.5%에서 46.6%로 5.1% 크게 향상시켰다. 이 연구는 의미론적 분할의 정확도가 향상됨에 따라 제안된 방법의 성능이 현저하게 향상될 잠재력이 있음을 경험적으로 보여주었다.

## 5. 강점, 한계

**강점**:

- **새로운 파이프라인**: 기존의 "탐지 후 분할" 방식이 아닌, 의미론적 범주 분할을 기반으로 인스턴스 분할을 수행하는 새로운 접근 방식을 제안하여 기존 방법론과 차별화된다.
- **온라인 부트스트래핑**: 훈련 중 어려운 픽셀에 집중하게 함으로써 네트워크의 학습 효율성을 높이고, 편향된 훈련 데이터 문제를 완화하는 효과적인 온라인 부트스트래핑 방법을 도입했다.
- **최고 성능 달성**: PASCAL VOC 2012 데이터셋에서 의미론적 분할에서 당시 최고 수준의 평균 IoU를 달성했으며, 인스턴스 분할에서도 기존 최고 성능과 동등하거나 우수한 결과를 보였다.
- **잔여 네트워크 활용**: 깊은 잔여 네트워크를 활용하여 강력한 특징 학습 능력을 기반으로 높은 성능을 달성했다.

**한계**:

- **간단한 클러스터링**: 인스턴스를 찾기 위해 간단한 NMS(비최대 억제) 접근 방식을 사용하는데, 논문에서도 더 정교한 클러스터링 방법이 결과 개선으로 이어질 수 있다고 언급한다. 이는 개선의 여지가 있는 부분이다.
- **두 네트워크의 개별 훈련**: 의미론적 분할 네트워크와 지역화 네트워크를 개별적으로 훈련하는 방식은 전체 시스템의 최적화에 제한을 줄 수 있다.
- **Hough 변환과의 연결성**: 일반화된 Hough 변환 기반 접근 방식과의 연결성을 언급하지만, 그 깊이 있는 분석이나 새로운 이론적 기여는 명확히 제시되지 않는다.

## 6. 결론

이 논문은 범주 수준 의미론적 분할을 활용하여 인스턴스 수준 이미지 분할을 수행하는 효과적인 새로운 프레임워크를 제시한다. 주요 기여는 혁신적인 파이프라인 설계, 온라인 부트스트래핑 훈련 방법론, 그리고 깊은 완전 컨볼루션 잔여 네트워크의 최적화에 있다. PASCAL VOC 2012 데이터셋에서 최고 수준의 의미론적 분할 성능과 경쟁력 있는 인스턴스 분할 성능을 달성함으로써, 이 연구는 의미론적 분할의 발전을 인스턴스 분할 문제에 성공적으로 연결시키는 중요한 발걸음을 내딛었다. 이 연구는 향후 이미지 이해 및 컴퓨터 비전 분야에서 인스턴스 분할을 위한 더욱 효율적이고 정확한 모델 개발에 중요한 기반이 될 가능성이 있다。
