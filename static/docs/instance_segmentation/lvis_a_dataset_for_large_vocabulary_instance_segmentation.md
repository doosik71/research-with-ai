# LVIS: A Dataset for Large Vocabulary Instance Segmentation

- **저자**: Agrim Gupta, Piotr Dollár, Ross Girshick
- **발표연도**: 2019
- **arXiv**: https://ar5iv.labs.arxiv.org/html/1908.03195v2

## 1. 논문 개요

이 논문은 대규모 어휘 인스턴스 분할(Large Vocabulary Instance Segmentation)을 위한 새로운 데이터셋인 LVIS(‘el-vis’로 발음)를 소개한다. 객체 탐지 연구의 발전은 미해결 과제에 연구 커뮤니티의 관심을 집중시키는 데이터셋에 의해 주도되어 왔다. 이 과정은 단순한 이미지에서 복잡한 장면으로, 바운딩 박스에서 분할 마스크로 연구의 초점을 옮겼다. LVIS는 164,000개의 이미지에 걸쳐 1,000개 이상의 엔트리 레벨 객체 카테고리에 대한 약 2백만 개의 고품질 인스턴스 분할 마스크를 수집하는 것을 목표로 한다. 자연 이미지에서 카테고리의 지프 분포(Zipfian distribution)로 인해 LVIS는 훈련 샘플이 적은 긴 꼬리(long tail) 카테고리를 자연스럽게 포함한다. 객체 탐지를 위한 최첨단 딥러닝 방법들이 저샘플(low-sample) 환경에서 제대로 작동하지 못한다는 점을 고려할 때, 이 데이터셋은 중요하고 흥미로운 새로운 과학적 도전 과제를 제시한다.

## 2. 핵심 아이디어

이 논문의 핵심 아이디어는 대규모 어휘 인스턴스 분할 연구를 위한 벤치마크 데이터셋인 LVIS를 설계하고 수집하는 것이다. LVIS는 1,000개 이상의 객체 카테고리를 다루며, 이는 기존 데이터셋(예: COCO의 80개 카테고리)보다 훨씬 많은 수이다. 특히, 자연 이미지에서 자주 나타나는 긴 꼬리 분포의 카테고리(즉, 훈련 샘플이 적은 희귀 카테고리)에 대한 인스턴스 분할 성능 평가를 가능하게 하는 데 중점을 둔다.

이를 위해 논문은 "평가 우선 설계 원칙(evaluation-first design principle)"을 따른다. 이는 데이터 수집 전에 평가 방법과 방식을 정확하게 정의하는 것을 의미한다. 또한, 여러 카테고리가 중첩될 수 있는 상황(예: "장난감 사슴"이 "장난감"과 "사슴" 모두에 해당)에서 공정한 평가를 위해 "연합 데이터셋(federated dataset)" 개념을 도입한다. 연합 데이터셋은 각 카테고리별로 작은 구성 데이터셋의 연합으로 이루어져, 각 카테고리에 대해 모든 인스턴스가 주석 처리된(exhaustive annotation) 긍정적 집합($\mathcal{P}_c$)과 해당 카테고리가 존재하지 않는 부정적 집합($\mathcal{N}_c$)을 보장한다. 이는 주석 작업 부하를 크게 줄이면서도 공정한 평가를 가능하게 한다.

## 3. 상세 방법 설명

LVIS 데이터셋은 고품질의 분할 마스크를 생성하기 위한 크라우드소싱 주석 파이프라인을 사용한다. 이 파이프라인은 기계 학습 알고리즘을 사용하지 않고, 주석자들이 이미지 내에서 객체를 발견하는 반복적인 과정을 통해 긴 꼬리 카테고리를 발견한다. 이 방법은 데이터 라벨링을 자동화하는 데 기계 학습 알고리즘을 사용하는 것을 피한다.

데이터셋 설계의 핵심은 "연합 데이터셋(federated dataset)" 개념이다. 각 카테고리 $c$에 대해, 전체 데이터셋 $\mathcal{D}$ 내에 두 개의 서로 다른 부분 집합이 존재한다:

- **긍정적 집합($\mathcal{P}_c$)**: 이미지의 부분 집합으로, 이 집합 내의 모든 $c$ 인스턴스가 분할된다. 즉, $\mathcal{P}_c$는 카테고리 $c$에 대해 철저하게 주석 처리된다.
- **부정적 집합($\mathcal{N}_c$)**: 이미지의 부분 집합으로, 이 집합 내의 어떤 이미지에도 $c$ 인스턴스가 나타나지 않는다.

카테고리 $c$에 대한 $\mathcal{P}_c \cup \mathcal{N}_c$를 사용하여 표준 COCO 스타일 AP(Average Precision) 평가를 수행한다. 알고리즘은 각 이미지에서 모든 카테고리에 대해 예측을 수행해야 하며, 평가 오라클은 각 카테고리를 해당 구성 데이터셋($\mathcal{P}_c \cup \mathcal{N}_c$)에서 공정하게 평가한다. 이러한 설계는 다중 유효 라벨 문제를 해결하고 주석 작업량을 크게 줄이는 데 기여한다.

## 4. 실험 및 결과

논문은 5,000개의 이미지로 구성된 `val` 부분 집합에 대한 상세 분석 결과를 보고하며, 이는 두 번 주석 처리되었다. 또한, 총 77,000개의 이미지가 추가로 주석 처리되었으며, 이는 최종 데이터셋의 약 50%에 해당한다. 이 버전을 LVIS v0.5라고 부른다. LVIS v0.5를 기반으로 한 첫 번째 LVIS Challenge는 ICCV 2019의 COCO 워크숍에서 개최될 예정이었다.

품질 측면에서, LVIS의 분할 마스크는 전문 주석가들과 비교했을 때, COCO 및 ADE20K 데이터셋보다 높은 중첩 및 경계 일관성을 보인다. 이는 LVIS의 주석 파이프라인이 고품질의 분할 마스크를 수집할 수 있도록 설계되었기 때문이다. (이 보고서에 제공된 내용에는 특정 정량적 결과 수치가 포함되어 있지 않으므로, 일반적인 품질 우위를 언급한다.)

## 5. 강점, 한계

**강점:**

- **대규모 어휘 지원**: 1,000개 이상의 카테고리를 포함하여 기존 데이터셋의 한계를 넘어 대규모 객체 인스턴스 분할 연구를 가능하게 한다.
- **긴 꼬리 분포 해결**: 자연 이미지의 지프 분포 특성상 발생하는 희귀 카테고리(few-shot regime) 문제를 연구할 수 있는 벤치마크를 제공한다.
- **고품질 주석**: 크라우드소싱 방식에도 불구하고 높은 중첩도와 경계 일관성을 가진 고품질의 분할 마스크를 제공하여, 미세한 알고리즘 성능 차이를 평가할 수 있게 한다.
- **공정한 평가 프로토콜**: "연합 데이터셋" 설계와 평가 우선 원칙을 통해 카테고리 간의 중첩 관계나 동의어 문제를 고려한 공정한 평가 방법을 제시한다.
- **COCO와의 연속성**: COCO와 동일한 인스턴스 분할 작업 및 AP 지표를 채택하여 기존 연구자들의 진입 장벽을 낮춘다.

**한계:**

- **저샘플(low-sample) 성능 문제**: 논문은 긴 꼬리 카테고리의 저샘플 환경에서 최첨단 딥러닝 방법들이 제대로 작동하지 못한다는 문제를 제기하며, LVIS가 이를 위한 도전 과제를 제시하는 것이 목표이다. 따라서 LVIS 자체는 이 문제를 해결하는 방법론이 아닌, 문제를 연구할 수 있는 기반을 제공한다.
- **데이터셋의 불완전성**: 논문 발행 당시 LVIS는 v0.5 단계로, 전체 데이터셋 수집이 완료되지 않은 상태이다 (약 50% 완료). 이는 초기 연구에는 유용하지만, 완전한 잠재력을 발휘하기 위해서는 전체 데이터셋의 완성이 필요하다.

## 6. 결론

LVIS: A Dataset for Large Vocabulary Instance Segmentation 논문은 대규모 어휘 인스턴스 분할 연구의 중요한 발전을 위한 새로운 데이터셋인 LVIS를 제시한다. 이 데이터셋은 1,000개 이상의 엔트리 레벨 객체 카테고리와 특히 희귀한 긴 꼬리 카테고리에 대한 고품질의 인스턴스 분할 마스크를 제공하여, 저샘플 환경에서의 객체 탐지 및 분할 알고리즘의 한계를 극복하기 위한 새로운 과학적 도전 과제를 제시한다. "평가 우선 설계 원칙"과 "연합 데이터셋"이라는 혁신적인 접근 방식을 통해, LVIS는 다중 유효 라벨 문제를 해결하고 주석 작업 부하를 줄이면서도 공정한 벤치마크 평가를 가능하게 한다. 이 연구는 컴퓨터 비전 커뮤니티가 보다 포괄적이고 실제적인 이미지 이해 시스템을 개발하는 데 중요한 기반을 제공하며, 향후 대규모 어휘 인스턴스 분할 및 긴 꼬리 학습 연구에 큰 영향을 미칠 것으로 기대된다.
