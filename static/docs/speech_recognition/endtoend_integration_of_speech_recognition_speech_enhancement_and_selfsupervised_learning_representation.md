# End-to-End Integration of Speech Recognition, Speech Enhancement, and Self-Supervised Learning Representation

Xuankai Chang, Takashi Maekaku, Yuya Fujita, Shinji Watanabe (2022)

## 🧩 Problem to Solve

자동 음성 인식(Automatic Speech Recognition, ASR) 시스템은 지난 10년간 딥러닝 기술의 발전 덕분에 크게 진보하였다. 특히 종단 간(End-to-End, E2E) ASR 시스템은 CTC(Connectionist Temporal Classification), Transducer, 그리고 어텐션 기반 인코더-디코더와 같은 다양한 기술을 통해 발전해왔다. 이러한 기술들은 깨끗한(clean) 음성 환경에서는 강력한 성능을 보여주지만, 현실 세계에서 녹음되는 예측 불가능한 잡음(noise)과 잔향(reverberation)이 포함된 시끄러운(noisy) 환경, 특히 단일 채널(monaural) 조건에서는 여전히 성능 저하가 발생하는 문제가 있다. 실제로 DNN-HMM 하이브리드 ASR 시스템이 E2E ASR 시스템보다 잘 알려진 CHiME-4 코퍼스와 같은 잡음이 많은 음성 코퍼스에서 더 나은 성능을 보이는 경우가 있다.

이러한 잡음 음성 인식 문제를 해결하기 위한 기존 접근 방식은 크게 두 가지이다. 첫째는 ASR 모델 자체를 잡음에 강인하게 훈련하는 것이고, 둘째는 ASR 모델에 입력하기 전에 전용 모델을 사용하여 잡음 음성의 명료도를 향상시키는 음성 향상(Speech Enhancement, SE) 또는 노이즈 제거(denoising) 전처리 기법을 사용하는 것이다. 하지만 단일 채널 음성 향상 기술은 필연적으로 음성 신호에 왜곡(distortion)을 발생시켜 ASR 성능을 저해할 수 있다는 문제점이 있다.

최근 자기지도 학습 표현(Self-Supervised Learning Representation, SSLR)은 ASR 성능 향상에 큰 잠재력을 보여주었지만, 현재 SSLR 모델의 주된 단점은 사전 훈련(pre-training) 비용이 매우 높다는 점이다. 또한, 기존의 Wav2Vec2.0이나 HuBERT 같은 SSLR 모델은 주로 깨끗한 음성 데이터(예: Libri-Light)로 사전 훈련되어, 잡음이 있는 실제 환경에서의 성능 향상에는 한계가 있었다. 특히, 다중 채널(multi-channel) 데이터에 비해 단일 채널의 잡음이 더 심한 경우 성능 개선이 제한적이었다.

이 논문은 잡음이 많은 환경, 특히 단일 채널 환경에서 강인한 음성 인식을 달성하기 위한 종단 간(E2E) ASR 모델을 제안하여 위에서 언급된 문제들을 해결하고자 한다. 논문의 목표는 음성 향상 모듈과 자기지도 학습 표현 모듈을 통합한 새로운 ASR 모델을 개발하고, 효율적인 학습 방식을 통해 잡음이 많은 환경에서 최첨단 성능을 달성하는 것이다.

## ✨ Key Contributions

이 논문의 핵심 기여는 다음과 같다.

1. **SE, SSLR, ASR의 종단 간(End-to-End) 통합 모델 제안**: 잡음에 강인한 음성 인식을 위해 음성 향상(SE) 모듈, 자기지도 학습 표현(SSLR) 모듈, 그리고 자동 음성 인식(ASR) 모듈을 단일 종단 간 모델인 IRIS(Integraded speech Recognition with enhanced speech Input for Self-supervised learning representation)로 통합하였다. 이는 기존 모델들이 SE와 ASR을 분리하여 훈련하거나, SSLR을 단순히 특징 추출기로만 활용하는 방식의 한계를 극복한다.
2. **효율적인 학습 방식 구축**: 제안된 IRIS 모델을 효과적으로 훈련하기 위해 각 모듈(SE, SSLR, ASR)을 사전 훈련된 모델로 초기화한 후, SE 및 ASR 모듈의 파라미터를 음성 향상 손실과 ASR 손실을 모두 사용하여 공동으로 미세 조정(fine-tuning)하는 방식을 확립하였다. 특히, SSLR 모듈의 파라미터는 미세 조정 과정에서 고정시켜 계산 비용을 절감하고 안정적인 학습을 유도한다. 이 미세 조정 전략은 개별 모듈을 단순히 연결했을 때 발생하는 성능 저하 문제를 해결하는 데 결정적인 역할을 한다.
3. **WavLM 기반 SSLR 모듈 활용의 검증**: 잡음에 강인한 SSLR 모델인 WavLM을 활용하여 시끄러운 환경에서의 특징 추출 성능을 극대화하였다. WavLM은 잡음/겹침 음성 데이터로 사전 훈련되었기 때문에 기존의 깨끗한 음성으로 훈련된 SSLR 모델보다 잡음 환경에서 더 우수한 성능을 제공한다.
4. **단일 채널 CHiME-4 벤치마크에서 최첨단 성능 달성**: 단일 채널 CHiME-4 ASR 태스크에서 제안하는 IRIS 모델이 기존의 최첨단 시스템(하이브리드 ASR 모델 기반)을 능가하는 성능을 달성하여, 잡음이 많은 단일 채널 환경에서 강인한 음성 인식의 새로운 기준을 제시하였다.

## 📎 Related Works

논문은 ASR 분야의 발전을 이끈 다양한 관련 연구들을 소개하고, 제안하는 IRIS 모델이 기존 연구의 한계를 어떻게 극복하는지 설명한다.

**기존 ASR 시스템:**

* **DNN-HMM 하이브리드 시스템**: 심층 신경망(DNN)을 사용하여 프레임 정렬 상태(예: 문맥 종속 음소)를 예측하는 방식이다. 이 모델은 초기 ASR의 발전에 크게 기여했으나, 잡음이 많은 환경에서는 E2E 시스템보다 여전히 강인한 성능을 보이는 경우가 있다.
* **종단 간(End-to-End, E2E) ASR 시스템**: CTC, Transducer, 어텐션 기반 인코더-디코더와 같은 기술을 포함하며, 전체 음성-텍스트 변환 과정을 단일 신경망으로 처리한다. 깨끗한 조건에서 강력한 성능을 보이지만, 잡음이 심한 환경, 특히 단일 채널의 경우 성능 저하가 발생한다.

**잡음 음성 인식 해결을 위한 기존 접근 방식:**

* **잡음에 강인한 ASR 모델 훈련**: ASR 모델 자체를 잡음에 강인하게 만드는 방식이다.
* **음성 향상(Speech Enhancement, SE) 전처리**: ASR 모델에 입력하기 전에 전용 모델로 잡음 음성의 명료도를 향상시키는 방식이다. SE 모델과 ASR 모델은 개별적으로 훈련되거나 공동으로 훈련될 수 있다. 그러나 단일 채널 음성 향상 기술은 왜곡을 발생시켜 ASR 성능을 저해할 수 있다는 것이 잘 알려진 사실이다.

**자기지도 학습 표현(Self-Supervised Learning Representation, SSLR):**

* **잠재력 및 한계**: SSLR은 ASR 성능 향상에 큰 잠재력을 보여주었지만, 사전 훈련 비용이 매우 높다. 또한, Wav2Vec2.0이나 HuBERT와 같이 깨끗한 음성 코퍼스(예: Libri-Light)로 사전 훈련된 모델은 잡음 조건에서의 성능 개선이 제한적이다. 이전 연구([21])에서는 이러한 모델을 특징 추출에 직접 사용하면 ASR 성능이 향상되었지만, 잡음 환경에서의 불일치(mismatch)로 인해 개선 폭이 작았다.
* **잡음 강인성 개선 노력**: WavLM은 시뮬레이션된 잡음/겹침 음성으로 표현 모델을 학습시켜 이 문제를 해결하려 했다. 또한, Wang et al.([25])은 Wav2Vec2.0을 기반으로 잡음 강인 SSLR 모델을 제안하여 CHiME-4에서 유망한 결과를 보였다.

**기존 접근 방식과의 차별점:**
본 논문은 기존 연구의 한계를 극복하기 위해 다음과 같은 차별점을 가진다.

* **종합적인 종단 간 통합**: 음성 향상(SE), 자기지도 학습 표현(SSLR), 그리고 ASR 모듈을 하나의 종단 간 모델로 통합하여 잡음 문제를 다룬다. 이는 SE의 왜곡 문제를 공동 훈련으로 완화하고, SSLR의 잡음 불일치 문제를 SE 모듈과 함께 해결한다.
* **효율적인 훈련 방식**: 각 모듈을 사전 훈련된 모델로 초기화하고, SE와 ASR 모듈을 미세 조정하며 SSLR 모듈은 고정하는 효율적인 학습 체계를 구축한다. 이는 높은 사전 훈련 비용과 복잡성을 줄이면서 성능을 극대화한다. 특히, 단순히 SE와 ASR을 연결했을 때 성능이 저하되는 기존의 문제점을 공동 미세 조정을 통해 해결한다.
* **WavLM의 전략적 활용**: 잡음에 강인하도록 설계된 WavLM을 SSLR 모듈로 채택하여, 잡음이 많은 입력에 대해서도 우수한 특징을 추출하도록 한다.

## 🛠️ Methodology

제안된 IRIS(Integraded speech Recognition with enhanced speech Input for Self-supervised learning representation) 모델은 잡음에 강인한 음성 인식을 목표로 음성 향상(SE) 모듈, 자기지도 학습 표현(SSLR) 모듈, 그리고 자동 음성 인식(ASR) 모듈을 종단 간(end-to-end)으로 통합한다. 이 모델은 모듈식 설계를 채택하여 각 모듈을 개별적으로 훈련한 다음 전체 모델을 미세 조정할 수 있다.

### 전체 파이프라인 또는 시스템 구조

IRIS 모델은 잡음이 섞인 입력 음성 $Y$를 받아 깨끗한 텍스트 $W$를 생성하는 과정을 다음과 같은 수식으로 표현할 수 있다.

$$
W = ASR(SSLR(SE(Y; \theta_{se}); \theta_{sslr}); \theta_{asr})
$$

여기서:

* $Y$는 잡음이 섞인 입력 음성 신호이다.
* $SE(\cdot; \theta_{se})$는 음성 향상(Speech Enhancement) 모듈로, 잡음 음성 $Y$를 향상된 음성 $Z$로 변환한다. $\theta_{se}$는 SE 모델의 파라미터를 나타낸다.
* $SSLR(\cdot; \theta_{sslr})$는 자기지도 학습 표현(Self-Supervised Learning Representation) 모듈로, 향상된 음성 $Z$로부터 특징 $F$를 추출한다. $\theta_{sslr}$는 SSLR 모델의 파라미터를 나타낸다.
* $ASR(\cdot; \theta_{asr})$은 자동 음성 인식(Automatic Speech Recognition) 모듈로, 추출된 특징 $F$로부터 텍스트 $W$를 예측한다. $\theta_{asr}$는 ASR 모델의 파라미터를 나타낸다.

이 파이프라인은 입력 잡음 음성이 SE 모듈을 통해 먼저 깨끗해지고, 이 깨끗해진 음성에서 SSLR 모듈이 효과적인 특징을 추출하며, 최종적으로 ASR 모듈이 이 특징을 기반으로 텍스트를 인식하는 구조를 가진다.

### 각 주요 구성 요소 및 역할

1. **음성 향상(Speech Enhancement, SE) 모듈**
    * **역할**: 입력 잡음 음성 $Y$에서 원치 않는 잡음과 잔향을 제거하고 음성 신호만 보존하여 향상된 음성 $Z$를 생성한다.
    * **수식**: $Z = SE(Y; \theta_{se})$
    * **구현**: Conv-TasNet [26] 모델을 SE 모듈로 사용한다. Conv-TasNet은 종단 간 시간 영역 음성 향상에 매우 성공적인 모델이다. 시간 영역 모델은 위상(phase) 문제를 고려할 필요가 없어 왜곡을 줄이는 데 도움이 될 수 있다.
    * **구성**: 상대적으로 작은 Conv-TasNet 모델을 사용하여 계산 비용을 절약한다. 인코더는 1-D 컨볼루션 레이어 1개(출력 채널 256, 커널 크기 40, 스트라이드 20)로 구성되며, 디코더는 역 1-D 컨볼루션 레이어를 사용한다. 분리(separation) 부분에서는 Temporal Convolutional Network (TCN)를 사용하며, 4개의 컨볼루션 블록이 2번 반복된다. 채널 수는 512, 커널 크기는 3이다. 병목(bottleneck)은 256채널을 가진다.

2. **자기지도 학습 표현(Self-Supervised Learning Representation, SSLR) 모듈**
    * **역할**: 음성 향상 모듈에서 나온 향상된 음성 $Z$로부터 ASR 태스크에 적합한 강력한 특징 $F$를 추출한다. 기존의 에너지 기반 특징(예: 로그 멜 필터뱅크, MFCC)을 대체하여 ASR 성능을 향상시킨다.
    * **수식**: $F = SSLR(Z; \theta_{sslr})$
    * **구현**: WavLM [24] 모델을 SSLR 모듈로 사용한다. WavLM은 HuBERT [20]와 유사하게 마스킹된 세그먼트의 의사 레이블(pseudo-labels)을 예측하도록 훈련되어 음성에서 언어 정보를 학습한다. 또한, 화자 식별, 분리, 화자 분리(diarization) 태스크에서 잡음을 처리하도록 학습되어 잡음에 강인하다.
    * **사전 훈련 데이터**: Libri-Light (60k 시간), GigaSpeech (10k 시간), VoxPopuli (24k 시간)를 포함하는 대규모 비레이블 음성 데이터로 사전 훈련되었다.
    * **특징 차원 감소**: SSLR 모델에서 추출된 특징은 일반적으로 차원이 높기(예: 1,024) 때문에, 선형 레이어를 사용하여 ASR 인코더에 입력하기 전에 128로 차원을 감소시킨다.

3. **자동 음성 인식(Automatic Speech Recognition, ASR) 모듈**
    * **역할**: SSLR 모듈에서 추출된 특징 $F$를 기반으로 최종 음성 텍스트 $W$를 생성한다.
    * **수식**: $W = ASR(F; \theta_{asr})$
    * **구현**: Joint CTC / 어텐션 기반 인코더-디코더 프레임워크 [5]를 사용하여 E2E-ASR 모델을 구축한다. 이 모델은 Transformer 블록을 기반으로 한다.
    * **구성**: ASR 모델은 12개의 인코더 Transformer 레이어와 6개의 디코더 Transformer 레이어로 구성된다. 각 Transformer 레이어는 4개의 어텐션 헤드를 가지며, 선형 프로젝션의 차원은 2,048이다. 인코더는 두 개의 컨볼루션 레이어를 사용하여 입력 특징 시퀀스를 다운샘플링하며, 총 프레임 시프트는 40ms이다. 드롭아웃은 0.1로 설정된다.

### 훈련 목표, 손실 함수, 추론 절차 또는 알고리즘 흐름

**훈련 절차 및 목표**:
IRIS 모델의 훈련은 다음과 같은 다단계 접근 방식을 따른다.

1. **개별 모듈 사전 훈련**:
    * **SE 모듈**: Conv-TasNet은 시뮬레이션된 잡음 음성만 사용하여 사전 훈련된다. 훈련 목표는 입력 잡음 음성과 참조 깨끗한 음성 간의 SI-SNR(Scale-Invariant Signal-to-Noise Ratio) 손실을 최소화하는 것이다. Adam 옵티마이저를 사용하며 학습률은 $1 \times 10^{-3}$이다.
    * **ASR 모듈**: Transformer 기반의 E2E-ASR 모델은 CHiME-4 훈련 세트의 다중 채널 잡음 발화와 깨끗한 WSJ0+WSJ1 발화를 사용하여 사전 훈련된다. 훈련 목표는 CTC 손실과 어텐션 손실의 조합이다. Adam 옵티마이저를 사용하며 피크 학습률은 $1 \times 10^{-3}$이고 20k 스텝 웜업(warm-up)을 거친다. 훈련 중 Fbank 및 SSLR 특징에 대해 SpecAugment [32]를 적용한다.
    * **SSLR 모듈**: WavLM 모델은 대규모 비레이블 음성 데이터(Libri-Light, GigaSpeech, VoxPopuli)로 독립적으로 사전 훈련된다. 이 모델의 파라미터는 IRIS 모델의 미세 조정 단계에서 업데이트되지 않고 고정된다.

2. **IRIS 모델 미세 조정(Fine-tuning)**:
    * **초기화**: 모든 세 모듈($\theta_{se}$, $\theta_{sslr}$, $\theta_{asr}$)은 사전 훈련된 모델의 파라미터($\hat{\theta}_{se}$, $\hat{\theta}_{sslr}$, $\hat{\theta}_{asr}$)로 초기화된다.
    * **업데이트**: 미세 조정 단계에서는 SE 모듈($\theta_{se}$)과 ASR 모듈($\theta_{asr}$)의 파라미터만 업데이트되며, SSLR 모듈($\theta_{sslr}$)의 파라미터는 고정된다. 이는 대규모 SSLR 모델을 전체적으로 훈련하는 데 드는 어려움을 해결하고 안정적인 학습을 돕기 위함이다.
    * **손실 함수**: 미세 조정은 음성 향상 손실(SI-SNR)과 ASR 손실(CTC + 어텐션)을 모두 사용하여 10 에포크 동안 수행된다. 이는 SE 모듈이 ASR 성능에 긍정적인 영향을 미치도록 훈련 목표를 정렬하는 데 필수적이다.
    * **옵티마이저**: ASR 모델 훈련과 동일한 Adam 옵티마이저를 사용하며, 학습률은 $5 \times 10^{-4}$이다.

**추론 절차**:
디코딩 시에는 미세 조정 과정에서 가장 좋은 정확도를 보인 10개의 체크포인트에 대해 모델 평균(model averaging)을 수행하여 최종 예측을 얻는다. 빔 서치(beam search) 시에는 문자 수준(character level) Transformer 언어 모델을 가중치 1.0으로 사용한다.

## 📊 Results

본 연구는 CHiME-4 코퍼스에서 단일 채널 ASR 태스크를 평가하여 IRIS 모델의 성능을 검증하였다. 평가는 시뮬레이션된(Simulated) 및 실제(Real) 음성 기록에 대한 개발(Dev) 및 테스트(Test) 세트에서 WER(Word Error Rate)을 측정하여 진행되었다.

### 데이터셋, 작업, 기준선, 지표

* **데이터셋**: CHiME-4 코퍼스 [8]를 사용한다. 이 데이터셋은 Wall Street Journal(WSJ0) 코퍼스에서 추출된 음성의 실제 및 시뮬레이션된 6채널 잡음 녹음본을 포함한다. 버스, 카페, 보행자, 거리의 네 가지 잡음 시나리오를 포함한다.
  * **훈련 데이터**: 1,600개의 실제 발화와 7,138개의 시뮬레이션된 발화(CHiME-4), 그리고 깨끗한 Wall Street Journal (WSJ0 + WSJ1) 발화가 ASR 모델 훈련에 사용된다. SE 모델 훈련에는 시뮬레이션된 발화만 사용된다.
  * **평가 데이터**: 단일 채널 개발 및 테스트 세트가 사용된다.
* **작업**: 잡음에 강인한 자동 음성 인식 (Robust Automatic Speech Recognition).
* **기준선**:
  * 기존 하이브리드 ASR 시스템 (Kaldi Baseline [33], Du et al. [34], Yang et al. [7]).
  * 기존 E2E ASR 시스템 (Wav2Vec-Switch [25]).
  * 본 연구에서 훈련한 E2E Transformer ASR 시스템 (Fbank, HuBERT, WavLM 특징 사용).
* **지표**: 단어 오류율 (Word Error Rate, %WER).

## 🧠 Insights & Discussion

### 논문에서 뒷받침되는 강점

* **강력한 잡음 강인성**: SE 모듈과 잡음에 강인한 SSLR(WavLM)을 종단 간으로 통합하고 효과적인 미세 조정 전략을 적용함으로써, 잡음이 심한 단일 채널 환경에서 기존의 최첨단 시스템들을 뛰어넘는 뛰어난 성능을 달성하였다.
* **모듈식 설계의 유연성**: SE, SSLR, ASR 모듈을 개별적으로 사전 훈련하고 통합하는 모듈식 설계는 각 구성 요소의 강점을 활용하면서 전체 시스템의 복잡도를 관리할 수 있게 한다. 이는 재사용성을 높이고 연구 확장을 용이하게 한다.
* **효과적인 공동 미세 조정**: SE 모델의 왜곡이 ASR 성능을 저해할 수 있다는 문제점을, SE와 ASR 모델을 공동으로 미세 조정함으로써 해결하였다. 이는 두 모듈의 훈련 목표를 정렬하여 시너지를 창출하는 데 핵심적인 역할을 한다. 특히, ASR 손실을 SE 모듈 훈련에 통합하는 것이 중요함이 입증되었다.
* **WavLM의 잠재력 활용**: 잡음 데이터로 사전 훈련된 WavLM을 SSLR 모듈로 활용하여, 깨끗한 음성 데이터로 훈련된 HuBERT 등의 모델이 가지는 잡음 환경에서의 불일치 문제를 성공적으로 극복하였다. 이는 도메인 불일치 문제를 해결하는 데 있어 잡음 강인한 SSLR 모델의 중요성을 강조한다.
* **효율적인 학습 스키마**: 거대한 SSLR 모델의 파라미터를 미세 조정 시 고정하고 SE 및 ASR 모듈만 업데이트하는 방식은 계산 비용을 절감하면서도 강력한 성능을 이끌어내어 실용적인 접근 방식을 제시한다.

### 한계, 가정 또는 미해결 질문

* **사전 훈련에 대한 의존성**: IRIS 모델은 각 모듈(SE, SSLR, ASR)이 미리 훈련된 모델로 초기화되어야만 성공적으로 훈련된다. 무작위 초기화부터 종단 간 공동 훈련은 수렴하지 못하는 것으로 나타났다. 이는 대규모 데이터와 컴퓨팅 자원을 필요로 하는 사전 훈련 단계의 중요성을 강조하지만, 동시에 초기화 없이 완전한 종단 간 학습을 가능하게 하는 추가적인 학습 기법이 필요함을 시사한다.
* **SSLR 모델의 크기 및 데이터 의존성**: WavLM과 같은 SSLR 모델은 매우 많은 파라미터를 가지고 있으며 방대한 비레이블 데이터로 사전 훈련된다. 이는 이러한 모델의 개발 및 활용에 높은 계산 비용과 데이터 접근성을 요구하며, 일반 연구 그룹에는 부담이 될 수 있다. 논문에서도 "사전 훈련된 SSLR 모델은 더 많은 파라미터와 더 많은 데이터를 사용한다"고 명시적으로 언급한다.
* **일반화 능력**: 제안된 시스템의 일반화 능력(generalization ability)에 대한 추가적인 탐색이 필요하다. CHiME-4 데이터셋 외의 다양한 잡음 환경이나 언어에 대한 성능 검증이 부족하다.
* **SE 모듈의 미세 조정 데이터 제한**: SE 모듈의 미세 조정이 시뮬레이션된 데이터에만 의존하는 경향이 있는데, 이는 실제 잡음 환경에서의 SE 성능을 더욱 최적화하는 데 한계가 될 수 있다.

### 논문에 근거한 간략한 비판적 해석 및 논의사항

이 논문은 잡음 음성 인식 분야에서 중요한 진전을 이루었다. 특히, 음성 향상(SE)과 자기지도 학습 표현(SSLR)을 종단 간 ASR 프레임워크에 효과적으로 통합하는 방법을 제시했다는 점에서 높은 평가를 받을 만하다. 기존 연구에서 SE가 ASR 성능을 저해할 수 있다는 문제를 지적했음에도 불구하고, 본 연구는 SE와 ASR 모듈의 공동 미세 조정이라는 영리한 전략을 통해 이러한 한계를 극복하고 ASR 성능을 크게 향상시켰다. 이는 모듈 간의 훈련 목표를 정렬하는 것이 얼마나 중요한지를 보여준다.

그러나 이 모델의 성공이 강력한 사전 훈련된 SSLR(WavLM) 모델과 각 모듈의 사전 훈련에 크게 의존한다는 점은 고려해야 한다. 이는 현재 딥러닝 연구의 일반적인 경향이지만, "진정한" 종단 간 시스템이 무작위 초기화부터 성공적으로 훈련될 수 있는가에 대한 질문은 여전히 남아있다. 논문에서 언급된 것처럼, 무작위 초기화부터의 훈련 실패는 아직 해결되지 않은 과제이며, 이를 해결한다면 모델의 유연성과 범용성이 더욱 증대될 것이다.

전반적으로, IRIS 모델은 잡음이 많은 환경에서 강인한 음성 인식을 위한 실용적이고 효과적인 접근 방식을 제시하며, 특히 단일 채널 시나리오에서 인상적인 결과를 보여주었다. 이는 ASR 시스템의 실제 적용 가능성을 한 단계 끌어올리는 데 기여하였다.

## 📌 TL;DR

이 논문은 잡음에 강인한 자동 음성 인식(ASR)을 위한 새로운 종단 간(E2E) 모델인 IRIS(Integraded speech Recognition with enhanced speech Input for Self-supervised learning representation)를 제안한다. IRIS는 음성 향상(SE) 모듈(Conv-TasNet), 자기지도 학습 표현(SSLR) 모듈(WavLM), 그리고 ASR 모듈(Transformer 기반 Joint CTC/Attention)을 단일 파이프라인으로 통합한다.

핵심 기여는 각 모듈을 사전 훈련된 모델로 초기화한 후, SE 및 ASR 모듈의 파라미터를 음성 향상 손실과 ASR 손실을 모두 사용하여 공동으로 미세 조정하는 효율적인 학습 방식에 있다. 특히 SSLR 모듈의 파라미터는 미세 조정 시 고정된다. 실험 결과, 이 전략은 단순히 SE와 ASR을 연결했을 때 발생하는 성능 저하 문제를 해결하고, SE 모듈의 왜곡 문제를 완화한다. WavLM은 잡음이 많은 데이터로 사전 훈련되어 기존 SSLR 모델의 잡음 환경에서의 한계를 극복하는 데 중요한 역할을 한다.

단일 채널 CHiME-4 태스크 평가에서 IRIS 모델은 실제 테스트 세트에서 3.92%의 WER을 달성하며 기존 최첨단 시스템을 능가하는 새로운 최고 성능을 기록하였다. 이는 심지어 CHiME-4 챌린지의 2채널 최고 결과와도 견줄 만한 수준이다.

이 연구는 잡음이 많은 환경에서 강인한 ASR 시스템을 구축하는 데 있어 SE와 SSLR의 종단 간 통합 및 공동 미세 조정이 매우 효과적인 접근 방식임을 보여준다. 이 연구는 실제 환경에서의 음성 인식 시스템 적용 가능성을 높이고, 향후 무작위 초기화부터의 완전한 종단 간 학습 탐구와 같은 후속 연구의 중요한 기반을 제공할 것으로 기대된다.
